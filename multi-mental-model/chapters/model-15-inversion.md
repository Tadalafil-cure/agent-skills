# 模型15：反向思考

**核心检验问题**：真正能杀死逻辑的是什么？找对刺客了吗？

**运用要点**：
1. 芒格最推崇的思维工具：反过来想，总是反过来想
2. 对于任何投资论断，问：「什么会杀死这个逻辑？」而非「为什么它对」
3. 列出 3-5 个真正能杀死逻辑的因素——限定为**基本面层面**，非情绪/资金
4. 判断当前叙事是否正确识别了「真正的刺客」
5. 叙事找错刺客 = 整个诊断方向错误

**诊断清单**：
- [ ] 论文的核心逻辑（做多或做空）如果要被杀死，真正的刺客是什么？
- [ ] 论文目前攻击/辩护的「刺客」是真正的那个，还是被叙事错置的？
- [ ] 真正杀逻辑的力量是否在时间表上（如 2027-2028 产能集中投放）？
- [ ] 当前叙事是否在帮多头逻辑续命（而非杀死它）？

**典型误判**：
- 把短期情绪冲击当成基本面的杀手
- 把帮助逻辑的因素误判为杀死逻辑的因素
- 没有区分「何时」——杀逻辑的力量可能远在未来

**案例**：「什么会杀死天赐材料？」→ 不是六氟短期价格方向（不可预测），而是六氟长期供需结构是否逆转 + 固态电池是否颠覆电解液路线
