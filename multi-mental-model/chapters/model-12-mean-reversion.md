# 模型12：均值回归

**核心检验问题**：回归杀的是估值/周期价格，还是产业逻辑？

**运用要点**：
1. **关键区分**：回归的对象是什么？
   - 估值回归（市盈率从极端回到正常）——通常是健康的
   - 周期价格回归（商品价格从高峰回落）——周期股宿命
   - 产业逻辑回归（行业被证伪）——这才是真正的杀逻辑
2. 历史极端值（如 190% 涨幅、72% 营业利润率）本身就是远离均值的信号
3. 高景气不会永远高景气——但回归的原因要定位准确
4. 真正触发回归的是**供给侧的产能投放时点**，而非叙事

**诊断清单**：
- [ ] 论文是拿回归来杀估值/周期，还是杀产业逻辑？
- [ ] 回归的触发条件是什么？时点是否明确？（如产能投放的具体时间）
- [ ] 论文是否把「回归杀的是 A」错误归因为「B 被证伪」？
- [ ] 历史数据的极端程度：是否是真正的异常值还是新常态？

**案例**：天赐材料净利润从 18.91 亿 → 4.84 亿 → 13.44 亿 → Q1 2026 单季 16.47 亿——极端波动是均值回归的典型
