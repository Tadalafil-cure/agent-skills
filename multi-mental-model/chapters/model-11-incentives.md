# 模型11：激励模型

**核心检验问题**：谁在喊、为什么喊？

**运用要点**：
1. 每个信息源都有激励结构——分析师、媒体、KOL、上市公司自身
2. **关键区分**：分析师的激励是维护客户关系（卖出评级会得罪上市公司和机构客户），基金经理的激励是实际回报
3. 「说」和「做」之间的裂口是最强信号——如 21 家分析师一致看多但基金减仓 40%
4. **也要检查自身激励**：诊断框架本身是否有固有偏见？（如天然奖励「找出市场漏洞」的论据）
5. 看清动机，能过滤掉一半噪音

**诊断清单**：
- [ ] 论文引用的每个信息源，其发布者的激励是什么？
- [ ] 「一致看好」中有多少来自利益相关方？
- [ ] 论文自身的分析框架是否有系统性偏见？
- [ ] 卖方的「说」和买方的「做」是否一致？
