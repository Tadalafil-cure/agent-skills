# 模型2：价格弹性

**核心检验问题**：是用高弹性市场的现象推断整个市场吗？

**运用要点**：
1. 区分市场细分：高弹性段（价格敏感、替代容易）vs 低弹性段（粘性强、切换成本高）
2. 垄断利润来自低弹性段——叙事若用高弹性段的现象推断整体定价权崩溃，是以偏概全
3. 在股票语境中：PE 单一维度的「折价」不能线性外推到「整体低估」——PB 等另维度可能指向相反方向
4. PE+PB 交叉验证是检测「价格弹性错位」的有效工具

**典型叙事断点**：
- 「价格战 → 全行业利润崩塌」→ 检查低弹性客户段的粘性
- 「便宜 → 买入」→ 检查另一估值维度是否「昂贵」

**案例**：天赐材料 PE 折价 29%（便宜）vs PB 溢价 77%（昂贵）——PE 维度上的便宜不能外推到整体低估
