# 模型5：能力圈

**核心检验问题**：论据来源在你能力圈内吗？

**运用要点**：
1. 清楚画定你的能力圈边界：哪些是你有数据/经验/专业判断的领域
2. 圈外的信息，不应成为下注决策的核心依据
3. **标注盲区是基本动作**——但更重要的是：盲区存在时降低结论强度
4. 「核心变量不可预测」应直接决定结论的天花板，而不只是标注后照常下判断
5. 诚实的能力圈判断会给出更接近 50/50 的概率，而非高确信度的单边结论

**典型错误**：
- 用圈外信息推导圈内结论（如用模型跑分推导半导体投资）
- 标注了盲区但不对结论强度做相应下调
- 多 agent 辩论框架可能自带「发现错误定价」的偏见——需反思框架本身的能力圈
