# 模型7：赛马赔率模型

**核心检验问题**：好故事 ≠ 好赔率，赌了几件事同时发生？

**运用要点**：
1. 把任何投资论断转译成「你在同时赌哪几件事发生」
2. 列出每件事的先验概率，乘起来 = 总赔率
3. 好故事 + 差赔率 = 不下注
4. 即使在看空/看多判断中，「不下注等验证」往往比下注更优
5. 核心变量（如商品价格方向）不可预测 → 赔率计算无基础 → 不下注

**诊断清单**：
- [ ] 论文的做多/做空赌注涉及哪几件事？
- [ ] 每件事的独立概率大约多少？
- [ ] 有没有一件事的概率接近 50/50（即赌博）？
- [ ] 「不下注」是否应该被列为选项？
