# 模型10：Lollapalooza 效应

**核心检验问题**：是多因素共振还是单一基本面恶化？

**运用要点**：
1. Lollapalooza = 多股独立力量同时同向作用，产生远超任何单一因素的效果
2. 极端行情本身说明这是情绪、资金、叙事的共振，而非单一基本面恶化
3. **反过来用**：共振涨上去的，也会共振跌下来——与工厂停产、订单取消毫无关系
4. 识别资金面的「深度分裂」——不同参与者的行为方向不一致时，价格由多力博弈决定

**诊断清单**：
- [ ] 当前价格/走势是哪些独立力量共振的结果？（情绪、资金、叙事、基本面、强制平仓……）
- [ ] 不同资金方（北向/融资/主力/机构/散户）的方向是否一致？
- [ ] 论文是否将多因素共振误读为单一基本面信号？
- [ ] 「共振」本身就削弱了任何基于价格走势的单因素判断

**案例**：天赐材料论文识别出融资多空比 274:1 + 换手率 8.46% + 21/21 看多 + 基金减仓 -40% → 四股独立力量共振
