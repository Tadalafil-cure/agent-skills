# 模型14：贝叶斯更新

**核心检验问题**：新证据该调多少概率？是否过度外推？

**运用要点**：
1. 每一条新证据应该让你**小幅**调整概率，而非从一个极端跳到另一个
2. 区分证据的**权重**：某些证据（如结构性供需数据）的权重远大于另一些（如一次跑分/单月数据）
3. 核心变量数据缺失本身也是证据——它应该下调任何依赖该变量的结论的确信度
4. 看到单一数据点就推翻整个先验 = 不是贝叶斯更新

**诊断清单**：
- [ ] 论文的先验（初始判断）是基于什么？确信度如何？
- [ ] 论文引用的每条证据，权重排序是什么？
- [ ] 核心变量数据缺失时，确信度是否对应下调？
- [ ] 论文是否设置了「新证据触发重新评估」的时间节点？
- [ ] 先验强度的校准：如果一个判断的支撑数据中有多项「缺失/不可预测」，先验应该是 50/50

**在诊断中的评价**：
- 贝叶斯结构合理 = 有明确的先验 + 证据权重分层 + 设置了更新节点
- 贝叶斯结构不足 = 先验过强、未分级证据权重、无更新机制
