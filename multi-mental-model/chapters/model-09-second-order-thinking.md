# 模型9：二阶思维

**核心检验问题**：市场做一阶反应，二阶推演指向哪里？

**运用要点**：
1. **一阶**：直接因果（A → B）。通常是市场正在定价的逻辑
2. **二阶**：一阶后果的后果（A → B → C）。是超额收益的来源
3. **三阶**：二阶被市场部分定价后的剩余（A → B → C → D）
4. 机会往往藏在一阶与二阶的落差里

**诊断步骤**：
- 识别市场共识 = 一阶思维是什么
- 论文的核心论点 = 几阶思维？是否完成了一阶→二阶的推演？
- 论文是否触及了「市场已部分定价了二阶」的三阶？如果承认对手论据有「强」项但未充分消化，说明三阶展开不足

**在诊断中的输出格式**：
```
一阶（市场共识）：___
二阶（论文论点）：___
三阶（论文触及但未充分展开）：___
```
