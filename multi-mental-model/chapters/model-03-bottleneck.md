# 模型3：瓶颈理论

**核心检验问题**：真正卡脖子的环节在哪？叙事是否错置了瓶颈？

**运用要点**：
1. 画出产业链：从上游原材料到终端客户，逐环节标注供需紧张度
2. 找到利润最高的环节——往往就是瓶颈所在
3. 检查叙事是否把瓶颈错置到了非关键环节
4. 瓶颈环节的议价权即使在其他环节被冲击时也不会松动

**对企业护城河的影响**：
- 瓶颈不应只识别单一商品价格——企业的配方能力、客户关系、全球产能布局可能构成超越商品价格的护城河
- 缺乏完整同业对比时，护城河判断需标注不确定性

**典型叙事断点**：
- 「X 价格下跌 → 公司利润崩塌」→ 检查公司是否是 X 的供给侧还是需求侧、是否有非价格壁垒
