# 模型8：冲浪模型

**核心检验问题**：真正的浪头在哪？站对了没？

**运用要点**：
1. 识别「技术大浪」——不是短期题材，而是持续多年的结构性趋势
2. 站对浪头的人 = 卖铲子的人（基础设施/关键瓶颈），不是被叙事波及的环节
3. 区分「周期性浪」（商品价格周期）和「结构性浪」（长期趋势）
4. 一个企业可以同时站在多道浪上——检查是否因过度聚焦短期周期浪而低估了长期结构浪

**诊断关键判断**：
- 论文判断企业站在哪道浪上？是否只看了周期浪？
- 即使周期浪退潮，企业的结构性浪位置是否仍在？
- 「卖铲子」的类比：即使下游价格波动，提供不可替代基础设施的企业持续受益

**案例**：天赐材料站在「六氟涨价周期浪」（短期），但也站在「新能源+储能长期趋势浪」（结构）——论文过度聚焦前者
