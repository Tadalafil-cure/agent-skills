## 数据格式规范

### DataFrame 统一列

| 列名 | 类型 | 说明 |
|:--|:--|:--|
| `date` | `datetime64[ns]` | 数据日期（年月日，日频为具体日期，月频统一为当月 1 日） |
| `indicator` | `str` | 指标名称（中文，如 "CPI同比"、"M2同比增速"） |
| `value` | `float64` | 指标数值 |
| `unit` | `str` | 单位（% / 亿元 / 亿美元 / 元/美元 / 指数） |
| `source` | `str` | 数据来源（akshare/NBS/SAFE/chinamoney） |

### 排序

**全部指标降序排列（最新在前）。** `date` 列从新到旧。

### 数据窗

D_fetch 默认 `years_back=3`（2023-01 至今）。部分指标天然短于三年。

## 数据密度分级

| 密度 | 指标 | 说明 |
|:--|:--|:--|
| 🟢 **充足** (≥3 点, 三年覆盖) | 22 个指标 | 全部月频 + GDP 年度 | 三年全覆盖 |
| 🟡 **年度** (3 点, 三年) | GDP同比增速 | NBS API 年度，2023=5.4%/2024=5.0%/2025=5.0% |
| 🔴 **单点** (1 条) | DR007 | 仅当日 1.3829%，历史序列待 Wind |

## 数据源速度对比

| 源 | 类型 | 首次延迟 | 批量吞吐 | 适用 |
|:--|:--|:--|:--|:--|
| **akshare** | HTTP API | <1s | 秒返 | CPI/PPI/M2/M1/社融/PMI/LPR/SHIBOR/房价/外汇 |
| **NBS API** | Playwright 无头浏览器 | ~3s (启动) | 4 次/批 | 工业/社零/固投/失业率/进出口/地产三项 |
| **SAFE** | HTTP POST | <1s | 秒返 | USDCNY 汇率 |
| **chinamoney** | Playwright 无头浏览器 | ~3s | 单次 | DR007 当日 |

**总耗时**：全量 23 指标 ~60-90s（其中 NBS Playwright 启动占 ~10s，akshare 秒返占主体）。

## 已知缺口

| 缺口 | 影响 | 替代方案 |
|:--|:--|:--|
| DR007 历史序列 | 无法做历史趋势分析 | 当日数据可做方向参考；历史需 Wind/Bloomberg |
| 信用利差 | 货币信用分析缺关键一环 | 需 Wind 终端 |
| 土地出让收入 | 房地产周期缺供给端 | 财政部月度数据 web_search 可补 |
| 政府杠杆率 | 债务分析缺总量锚 | CNBS/BIS 季度数据 web_search 可补 |
| 核心 CPI | 通胀分析缺剔能源食品口径 | NBS 有指标，待补 cid 映射 |
| 2023 年度公报 | 年度基线缺一年 | NBS API `GDP指数_上年` 可替代 |

## 使用示例

```python
from macro_data import D_fetch, D_summary

# 按需拉取
data = D_fetch('货币金融')        # 8 指标, ~25s
latest_m2 = data['M2同比增速'].iloc[0]  # 最新一行
print(f"M2: {latest_m2['value']}% ({str(latest_m2['date'])[:10]})")

# 全量汇总
all_data = D_fetch()
df = D_summary(all_data)          # 合并为一张总表
print(df['indicator'].value_counts())  # 各指标数据量
```
