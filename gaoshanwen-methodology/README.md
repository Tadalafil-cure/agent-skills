# 高善文·宏观分析思维方法论 v5.12

> 蒸馏自 108 篇公开文章 / 70 万字 | 2010-2024

## 这是什么

一个可用于 Hermes Agent 的 **Skill**。它包含高善文博士公开文章中体现的宏观分析思维方法论，经过系统化提炼后形成可执行的三层架构：

- **第一层：能力层** — 8 个心智模型 + 15 条决策启发式 + 7 条反模式 + 5 维度盲点
- **第二层：操作层** — D_fetch 数据中间层（23 指标/三源合一）+ GAOBO 两遍制 + Loop 动态缺口清单（至多 3 轮，每轮后重检方法论）+ Loop 重检强制 Checklist（GAOBO-2 前置闸门）+ 复盘 + 历史案例库 + 7 大主题模块 + 中国冲击 2.0 + 跨国比较框架
- **第三层：表达层** — 14 条表达操作规则 + 3 种输出模式（演讲/研报/评论）+ 比喻工具箱 + 表达反例原则 + 正文/附录分离

方法论已通过 32 篇文章的逆向验证，零方法论不自洽。

## 安装

```bash
cp -r gaoshanwen-methodology ~/.hermes/skills/
```

**首次使用前运行依赖检查：**

```bash
python3 ~/.hermes/skills/gaoshanwen-methodology/scripts/check_deps.py
```

自动检测并安装 `akshare`、`playwright`、`pandas`、`chromium`。已有环境秒过，新环境自动补齐。

## 使用

直接提问即可触发。Skill 根据问题自动选择输出模式：

| 模式 | 场景 | 触发 |
|:--|:--|:--|
| 🔴 演讲 | 策略会演讲+对话综合——面向大众，逻辑为骨、锋利为刃。正文 6,500-9,000 字 | 默认。直接提问即此模式 |
| A 研报 | 深度系统研究——全量方法论，10 段完整结构。正文 10,000-15,000 字 | "详细分析""写研报""系统研究""展开" |
| B 评论 | 轻量快评/观点——单刀直入。正文 2,500-4,000 字 | "怎么看""简答""一句话""点评" |

每次输出标题下方标注当前模式名称——「演讲」「研报」「评论」之一。

### 示例

```
怎么看当前房地产市场的调整？
```
（演讲模式，自动拉取数据 → 核心判断 + 传导链 + 资产含义）

```
系统研究中国通胀形势，写一份深度分析
```
（研报模式，D_fetch 全量三组数据 → 10 段完整研究结构）

### 输出

三种模式均生成 `.md` 报告文件，保存在 `reports/` 子目录下，文件名 `YYYY-MM-DD_主题.md`。文末强制标注「基于 gaoshanwen-methodology vX.X 生成，不代表高善文本人观点，不构成投资建议」。

## 数据中间层

- **23 个宏观指标**，分三组：实体经济（11）、货币金融（8）、房地产（4）
- **三源合一**：akshare（货币信用端）+ NBS API（实体端）+ SAFE（汇率端）
- **D_fetch()** 一键拉取，~90 秒完成
- **执行流程**：D_fetch 拉取基线数据 → GAOBO-1 部署方法论、产出缺口清单 → Loop 动态缺口清单（至多 3 轮，每轮后强制走重检 Checklist，缺口清零且无新缺口时停止）→ GAOBO-2 终稿
- **Loop 搜索**：百度搜索补充缺口，具体工具取决于环境配置
- 详见 `references/data-dictionary.md`

## 已知局限

- 适用于宏观经济分析，不适用于微观公司分析、短期择时、技术判断
- 6 个硬骨头指标需 Wind/Bloomberg：信用利差、土地出让、政府收支、政府杠杆率、PE估值、核心CPI
- 方法论素材覆盖 2010-2024，2025 年后的新文章未纳入
