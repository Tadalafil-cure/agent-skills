## 这是什么

一个可用于 Hermes Agent 的 **Skill**。它包含高善文博士公开文章中体现的宏观分析思维方法论，经过系统化提炼后形成可执行的三层架构：

- **第一层：能力层** — 8 个心智模型 + 15 条决策启发式 + 7 条反模式 + 5 维度盲点
- **第二层：操作层** — GAOBO 五步执行流程 + 数据处理规范 + 标准输出格式 + 复盘机制 + 7 个分主题分析模块
- **第三层：表达层** — 14 条表达操作规则 + 3 种场景模式（默认/A深度/B轻量）+ 随机化微调

方法论已通过 32 篇文章的逆向验证，零方法论不自洽。

## 安装

将整个 `gaoshanwen-methodology` 文件夹复制到 Hermes 的 skills 目录：

```bash
cp -r gaoshanwen-methodology ~/.hermes/skills/
```

如果使用 Hermes Agent profile 模式，放在对应 profile 的 skills 目录：

```bash
cp -r gaoshanwen-methodology ~/.hermes/profiles/<your-profile>/skills/
```

安装后无需重启，下次对话自动加载。

## 使用

直接提问即可触发。Skill 会根据问题自动选择场景模式：

| 模式 | 场景 | 触发 |
|:--|:--|:--|
| 🔴 默认 | 策略会演讲+对话综合——面向大众，核心逻辑完整 | 默认。直接提问即此模式 |
| A | 深度研报——全量方法论，10 段完整结构 | "详细分析""写研报""系统研究" |
| B | 轻量点评——快评/评论/随笔 | "怎么看""简答""一句话" |

### 示例

```
怎么看当前房地产市场的调整？
```

（自动进入默认模式，输出核心判断 + 竞争解释 + 资产含义 + 最大风险）

```
系统研究中国通胀形势，写一份深度分析
```

（自动进入 A 模式，输出 10 段完整研究结构）

## 已知局限

- 适用于宏观经济分析，不适用于微观公司分析、短期择时、技术判断
- 数据源需用户提供（v4.1 将加入数据采集机制）
- 方法论素材覆盖 2010-2024，2025 年后的新文章未纳入
