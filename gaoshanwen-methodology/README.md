> 蒸馏自 108 篇公开文章 / 70 万字 | 2010-2024

## 这是什么

一个可用于 Hermes Agent 的 **Skill**。它包含高善文博士公开文章中体现的宏观分析思维方法论，经过系统化提炼后形成可执行的三层架构：

- **第一层：能力层** — 8 个心智模型 + 15 条决策启发式 + 7 条反模式 + 5 维度盲点
- **第二层：操作层** — D_fetch 数据中间层（23 指标/三源合一）+ GAOBO 五步 + Loop 缺口补充 + 复盘 + 7 大主题模块
- **第三层：表达层** — 14 条表达操作规则 + 3 种场景模式（默认/A深度/B轻量）+ 对联弹性化

方法论已通过 32 篇文章的逆向验证，零方法论不自洽。

## 安装

```bash
cp -r gaoshanwen-methodology ~/.hermes/skills/
```

**首次使用前运行依赖检查：**

```bash
python3 ~/.hermes/skills/gaoshanwen-methodology/scripts/check_deps.py
```

自动检测并安装 `akshare`、`playwright`、`pandas`、`chromium`。已有环境秒过，新环境自动补齐。

## 使用

直接提问即可触发。Skill 根据问题自动选择场景模式：

| 模式 | 场景 | 触发 |
|:--|:--|:--|
| 🔴 默认 | 策略会演讲+对话综合——面向大众，核心逻辑完整 | 默认。直接提问 |
| A | 深度研报——全量方法论，10 段完整结构 | "详细分析""写研报""系统研究" |
| B | 轻量点评——快评/评论/随笔 | "怎么看""简答""一句话" |

### 示例

```
怎么看当前房地产市场的调整？
```
（默认模式，自动拉取数据 → 核心判断 + 竞争解释 + 资产含义 + 最大风险）

```
系统研究中国通胀形势，写一份深度分析
```
（A 模式，D_fetch 全量三组数据 → 10 段完整研究结构）

### 输出

三种模式均生成 `.md` 报告文件，保存在 `reports/` 子目录下，文件名 `YYYY-MM-DD_主题.md`。

## 数据中间层

- **23 个宏观指标**，分三组：实体经济（11）、货币金融（8）、房地产（4）
- **三源合一**：akshare（货币信用端）+ NBS API（实体端）+ SAFE（汇率端）
- **D_fetch()** 一键拉取，~90 秒完成
- 详见 `references/data-dictionary.md`

## 已知局限

- 适用于宏观经济分析，不适用于微观公司分析、短期择时、技术判断
- 6 个硬骨头指标需 Wind/Bloomberg：信用利差、土地出让、政府收支、政府杠杆率、PE估值、核心CPI
- 方法论素材覆盖 2010-2024，2025 年后的新文章未纳入
