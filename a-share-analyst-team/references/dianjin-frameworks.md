## P0 框架（已融入，有脚本支撑）

### 1. 换手率分档 + 盘子修正 → `volume_classifier.py`

- **融入 Agent**：C1（趋势派）、C2（反转派）
- **脚本**：`scripts/volume_classifier.py`
- **规则**：
  - 大盘股(>1000亿): <1%冷清 / 1-3%正常 / 3-5%活跃 / >5%过热
  - 中盘(200-1000亿): 中位数阈值
  - 中小盘(<200亿): <1%冷清 / 1-3%正常 / 3-7%活跃 / 7-15%高度活跃 / >15%游资接力
- **C1 解读**：冷清≠危险，对大盘是常态；结合趋势方向判断
- **C2 解读**：过热+高位=天量天价反转风险；冷清+低位=地量见地价

### 2. 量价配合四宫格 → `volume_price.py`

- **融入 Agent**：C1（趋势派）
- **脚本**：`scripts/volume_price.py`
- **矩阵**（9 宫格）：放量/平量/缩量 × 上涨/平盘/下跌
  - 放量上涨=✅健康 / 缩量上涨=⚠️背离 / 平量上涨=→正常
  - 放量下跌=❌恶化 / 缩量下跌=🔄衰竭 / 平量下跌=→偏弱
  - 放量平盘=⚡分歧 / 缩量平盘=→观望 / 平量平盘=→正常
- **阈值**：放量=vol>MA20×1.5, 缩量=vol<MA20×0.7

### 3. 筹码集中度四级表 → `concentration.py`

- **融入 Agent**：D（资金博弈师）
- **脚本**：`scripts/concentration.py`
- **五维判定**：
  | 指标 | 高度控盘 | 中度控盘 | 低度控盘 |
  |------|---------|---------|---------|
  | 股东户数变化 | 持续减少 | 波动不定 | 持续增加 |
  | 户均持股趋势 | 持续增加 | 相对稳定 | 持续减少 |
  | 十大股东占比 | >60% | 40-60% | <40% |
  | 机构持股占比 | >30% | 15-30% | <15% |
  | 换手率水平 | <2% | 2-5% | >5% |
- **综合判定**：≥3 项高度→高度控盘；≥3 项低度→低度控盘；其他→中度控盘
