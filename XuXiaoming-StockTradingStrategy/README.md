## 核心规则（铁律 + 震荡框架）

### 铁律（不可违反）

```
趋势为王 > 结构修边 > 序列辅助
```

> [20190210] 如果突破趋势，你就甭管结构了。**趋势为王，结构修边。**
> [20190812] 序列不是买入原则，结构和趋势才是。

### 裁决架构（v4.6.11 共振驱动）

```
主板裁决 = 五指数共振投票（上证+深证+创业板+沪深300+中证500）
  强共振_上升 → 持股    强共振_下跌 → 空仓
  偏共振_偏多 → 观望(偏多)   偏共振_偏空 → 观望(偏空)
  混合/分化   → 观望

科技裁决 = 科创50定方向 + 双创共振升降档
  共振(一致) → 维持科创50强度
  混合/分化(不一致) → 降档一级
  科技与主板独立裁决，互不压制
```

> 单指数裁决（上证/深证/创业板/科创50）保留为辅助参考，不冒充共振共识。

### 铁律推论：市况决定结构含义

| 市况 | 顶结构 | 底结构 |
|------|------|------|
| 趋势上行/偏多 | 修边：首次→警戒，二次→减仓 | 趋势为王，不抄底 |
| 趋势下行/偏空 | 冗余——趋势本身就说空 | 等趋势突破，底结构是前提 |
| 震荡 | 取决于仓位（有→按趋势规则修边，无→不管） | 唯一入场标准。BS≥2=试探(100%)，BS=1=假信号(38%) |

### 钝化分析层（v4.6.11 新增，不入裁决）

| 维度 | 方法 | 转化率 | 假信号 | 用途 |
|------|------|:--:|:--:|------|
| 日线底钝化 | 极值法 20d | 88-95% | 5-15% | 分析标注 |
| 分钟线顶钝化(90min) | 极值法 20d | 89-98% | 2-7% | **可靠** |
| 分钟线底钝化 | 极值法 20d | 0-54% | 46-100% | 需结合日线 |
