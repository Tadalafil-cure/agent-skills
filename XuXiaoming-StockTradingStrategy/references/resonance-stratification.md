# 共振时机分层

## 核心发现

**共振强度不能单独用——时机比强度重要。**

深证成指基准，五指数（上证/深证/创业板/HS300/中证500）共振。科创50 已移除（永远是最后一个，等它 = 浪费时间）。

## 强共振_上升 → 按"之前已单边天数"分层

|:--|:--:|------|:--:|------|

## 典型案例

### 2024-10-08 滞后共振陷阱
- 9/24-9/30 已涨 36%，10/8 才触发六指数全共振
- "混合→偏多→强共振" 扩散模式：共振越晚出现越危险
- 通道先突破 + 共振随后跟上 = 好信号；两者同时 = 假突破

### 2024-9-24 黄金混合
- "混合"模式（3↑3震）→ 正是最佳入场点
- 深证/创业/HS300 已突破，上证/中证500 在跟
- "混合"20日后 +0.9%，胜率 55.6%——比强共振好

## 共振信号优先顺序

1. 通道突破 + 混合共振 → 转变期，高价值（如 924）
2. 通道突破 + 共振滞后跟上（5-14天后）→ 确认信号
3. 通道突破 + 共振同时出现 → 警惕假突破
4. 通道突破 + 共振出现在大涨后 → 滞后共振，可能是顶
