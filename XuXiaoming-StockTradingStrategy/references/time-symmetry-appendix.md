## 一、哲学根基

徐小明将时间对称为市场的平衡机制——不是神秘主义，是大量无序个体买卖在整体上涌现的统计秩序。

> "对称性是一种美……大量无序的个体买卖在整体上出现的秩序与美。"（2019）
> "平衡是无处不在的，目前调整周期和上升周期没有到达平衡，没有办法判定调整已经结束。"（2022）

在方法论体系中的位置：

```
规则层（不可违反）：趋势为王 > 结构修边 > 序列辅助
分析层（提供概率优势，不独立触发操作）：时间对称 → 识别"敏感时间段"
操作层：结构形成 + 在时间敏感段内 → 确定性增强。仅有时间对称、无结构 → 不操作
```

## 二、四种对称类型

### 类型 I：浪型内对称
同级别浪型之间：1浪≈5浪，2浪≈4浪（1:1）

### 类型 II：推动-调整对称
调整周期 = 推动周期 × k，k ∈ {0.5, 0.618, 1.0}
急涨→调整时间短（空间换时间），缓涨→调整时间长（时间换空间）

### 类型 III：ABC 内对称
A浪 ≈ C浪，B浪通常 ≤ A浪

### 类型 IV：跨段直接对称
不同上升/下跌波段的直接时长比较。中轴 = 前一个同级波段的实际天数。

**关键区分**：同向对称 1:1（上升vs上升、下跌vs下跌），反向对称 0.5~0.618（上升vs调整）。

## 四、时空互换

空间和时间呈反比。急涨→调整时间短（空间代偿）；缓涨→调整时间长。

经典案例（2020）：35天上升仅对应5天调整——但5天的空间跌幅足以和35天涨幅形成对称。

## 六、核心速查

```

│   前一波段天数 = 中轴                     │
│   当前波段接近中轴 → 时间敏感窗口          │
│   窗口内出现结构 → 置信度+1               │
│   仅有时间、无结构 → 不操作               │
│                                          │

│                                          │
```

## 不采纳声明

时间对称不进入正式分析框架，原因：

1. **徐小明本人的边界**："时间对称不是标准，结构才是"——他自己不靠它做交易
2. **事后选择风险**：中轴是事后从历史中挑的，存在德州神枪手问题
3. **机械化困难**：段落提取方法（regime / 价格方向 / 人工划段）均无法稳定复现徐小明的计数
4. **过拟合天然倾向**：锚点选择、容差设定、段落切分每一步都有主观自由度
