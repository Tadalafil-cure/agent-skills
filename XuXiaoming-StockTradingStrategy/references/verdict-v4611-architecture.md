## 改动后（v4.6.11）

### 主板裁决：五指数共振投票

```python
五指数(上证+深证+创业板+沪深300+中证500) 各算 regime → 投票

强共振_上升(≥4多) → 持股
强共振_下跌(≥4空) → 空仓
偏共振_偏多(3多0空) → 观望(偏多)
偏共振_偏空(0多3空) → 观望(偏空)
混合/分化 → 观望
```

### 科技裁决：科创50为主 + 双创共振升降档

```python
科创50单指数裁决 定方向 → 双创共振(创业板+科创50) 定升降档

共振_偏多/偏空 (两人一致) → 维持科创50强度
混合/分化 (两人不一致) → 降档一级

降档映射:
  持股 → 持股(警戒)
  持股(警戒) → 观望(偏多)
  观望(偏多) → 观望
  空仓 → 观望(偏空)
  ...
```

### 关键设计决策

| 决策 | 原因 |
|------|------|
| 科技不依赖五指数 | 科技与主板分化是常态，双市场架构的本意 |
| A16 从裁决逻辑移除 | 双创偏多不应因大盘混合而沉默 |
| 单指数裁决保留为参考列 | 深证低9等信号仍有分析价值，但不冒充共振共识 |
| reason 列显示票数 | `主板:五指数混合(0多0空)` |
