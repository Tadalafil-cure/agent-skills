## 逻辑解释："衰竭→最后一跌→反转"序列

```
高CHOP(横盘衰竭) → CHOP下降(最后一跌/恐慌) → 底部结构形成 → 反转
```

CHOP 下降 = 从震荡/衰竭状态重新加速下跌。这不是趋势延续，而是"恐慌出清"——结构在恐慌中形成。

## 典型案例：2022-04-27（+5.1%）

CHOP 轨迹：69 → 37 → 34 → 47 → 48 → 47 → 40 → **33**

- CHOP 从 69（极端衰竭）摔到 33（恐慌加速）
- 先衰竭到极致，再恐慌出清，然后结构确认
- 这是最干净的"恐慌底"信号

## 反例：2022-09-29（-4.7%）

CHOP 轨迹：63 → 58 → 50 → 47 → 38 → 43 → **37**

- CHOP 也是一路下降，但 R²=0.75——极干净的单边下跌
- 不是"衰竭后恐慌"，而是"一直没有衰竭的流畅下跌"
- 结构形成时跌势未止，继续跌 4.7%

**关键区分**：要看 CHOP 是否**先高后低**。纯粹的下降（从未高过）是趋势延续，不是恐慌底。

## 两种底类型的 CHOP 特征

| 底类型 | CHOP 特征 | 案例 |
|--------|----------|------|
| **恐慌底** | 先极端高(>60)→再暴跌(低CHOP<40) | 2022-04-27 |
| **磨底** | 长期45-55中性区间反复 | 2024-08/09 (924前夕) |

## CHOP 在底部结构中的价值

- **不作为独立入场条件**——CHOP 高/低本身对底部结构的区分力不够
- **作为恐慌底的确认信号**——前60日出现过 CHOP>60 + 结构形成时 CHOP<40 = 高质量恐慌底
- **作为趋势延续的排除信号**——CHOP 持续下降且 R²>0.5 = 流畅下跌，结构可能是假的

## H12 集成建议

```
下行趋势 + 底部结构形成 + 深跌(>15%) + 簇=1 + 钝化2-3天 + 前60日CHOP曾>60 → 试探
```

加上 CHOP 曾>60 的过滤后，2022-09-29（CHOP 最高63但整体趋势性下降 R²=0.75）被排除。
