# 结构是过程不是单点 · 924 案例

> 2026-07-11 用户纠正："那你对于底部结构的认知会不会太狭隘了？之前我们探讨924，再怎么样也是可以进的，这个不算底部？"

## 核心纠正

**结构形成是时间段，不是触发日。** 把"首次 bottom_structure=1"当买入信号是机械化的错误。

## 924 案例：2024年Q3 底部结构全记录

深证成指 2024年7月-9月，底部钝化+形成交替出现 20+ 次：

```
2024-07-01  8899 底部钝化=1 底部形成=1  ← 钝化开始
2024-07-05  8695 底部钝化=1 底部形成=1
2024-07-09  8705 底部钝化=1 底部形成=1
2024-07-24  8493 底部钝化=1              ← 价格持续新低
```

## 关键发现

1. **924 当天没有结构信号** — 底部形成在 9/19 就确认了。924 是趋势通道的突破日，不是结构日。

2. **7月-9月是一个"建仓区间"** — 钝化从未消失，价格从 8900→7992，期间底部结构反复确认。不是"哪天买"的问题，是整个区域都可以建仓。

3. **徐小明原文对得上**："买入的原则要么突破趋势，要么底部结构形成" — 底部结构在 Q3 反复形成，9/24 趋势突破确认。两者都在说同一个底部。

4. **结构"必须走完"** — 从 2024-07-01 首次底部形成，到 2025-07-02 首次顶部形成，结构走了整整一年，深证成指从 8899→10413（+17%）。

## 教训

- **不要把结构当单日信号** — 底部结构形成→钝化不消失→价格新低→再次形成→再次形成…这是结构级别不断加大的过程。
- **"中间不重要"成立** — 924 之后到顶部结构形成之前（2024-10 到 2025-07），中间所有波动"不改变操作结果"。
- **趋势和结构是并列入场条件** — 不要搞成"先结构后趋势"的串行逻辑。原文："要么突破趋势，要么底部结构形成。"
