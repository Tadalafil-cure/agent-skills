# 创业板+科创50 双共振回测

> 2026-07-12 | v4.5.0 新增功能验证

## 设计意图

科创50 + 创业板双共振用于捕捉科技分化的独立行情。这是基于徐小明方法论框架的扩展（非原文直接支撑），核心假设：双创指数的共振可能比五指数共振更早/更独立地反映科技板块方向。

## 关键发现

### 1. 共振_偏多 有效但不独立
- 84% 的"双创转偏多"发生在六指数已确认为强共振_上升之后
- 双创共振是**确认信号**，不是**预测信号**
- 924 后独立性提升：36% 的转多不依赖六指数先行（全量仅 16%）

### 2. 共振_偏空 无效
- 全量胜率 47%（接近随机）
- 924 后虽然数字好看（+5.10%, 90%胜率），但仅有 20 天样本，不可信
- 不能用于做空信号

### 3. 震荡期是最大陷阱
- 1024-0225 段：双创说了 25 天"偏多"，但 20 日后平均亏 -2.0%
- 这个阶段六指数共振是"混合"——说明震荡市中双创共振不可信
- **必须加过滤：共振_偏多 + 六指数非震荡 → 可信；共振_偏多 + 六指数震荡 → 沉默**

### 4. 2021年1月抱团瓦解是最大失效集群
- 连续 7 个交易日共振_偏多，但 20 日后 CYB 跑输 SH -5.6%~-10.5%
- 六指数同时显示强共振_上升——两个共振系统同时失效
- 这种"系统性谎言"无法用规则规避，属于黑天鹅

## 使用建议

| 场景 | 动作 |
|------|------|
| 共振_偏多 + 六指数强/偏多 | **升档确认**，科技股超额收益可靠 |
| 共振_偏多 + 六指数震荡/混合 | **沉默**，不据此修改裁决 |
| 共振_偏空 | **忽略**，不提供有效信号 |
| 双创转多领先六指数转多 | 关注，但不等同于确认（36% 独立率不足以独立触发） |

## 诚实边界

- 这不是徐小明原文理论，是基于方法论框架的扩展
- 回测路径依赖：924 后行情特殊（高波动+科技主导），早期数据可能不具代表性
- 共振_偏空及分化状态样本不足（20天/17天），结论可靠性低
