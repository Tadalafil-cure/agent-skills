# A股指数实时行情数据源

> v4.6.3 | 2026-07-13

## 可用接口

### 1. Sina 源（推荐用于实时行情）

```python
import akshare as ak
df = ak.stock_zh_index_spot_sina()
```

- 一次请求返回全部指数（约 1000+ 条）
- 无频率限制
- 覆盖六指数：上证(sh000001)、深证(sz399001)、创业板(sz399006)、科创50(sh000688)、沪深300(sh000300)、中证500(sh000905)
- 字段：最新价、涨跌幅、今开、最高、最低、昨收、成交量、成交额
- 响应时间：~2 秒（含 8 页分页）

### 2. 东方财富源（数据更全但有频率限制）

```python
df = ak.stock_zh_index_spot_em()
```

- 字段含量比、振幅
- 有频率限制，连续请求会被封
- 不推荐盘中高频使用

## 盘中裁决集成

`data_layer/realtime.py` 使用 Sina 源：

1. `get_spot()` — 拉取六指数实时行情
2. `estimate_chop_14()` — 用盘中最高/最低/最新价 + 历史13天日线估算CHOP(14)
3. `estimate_regime()` — 用最新价代理收盘，20日动量判断方向
4. `check_minute_structure()` — 拉取今日分钟线 + 运行结构引擎
5. `generate_briefing()` — 生成七段式盘中简报

## 关键设计决策

- **时区锁定北京时间**（UTC+8）：服务器可能在任意时区，`_china_now()` 锁定
- **交易日判断**：仅用 weekday + 交易时段（9:30-11:30, 13:00-15:00），不查节假日日历
- **盘前/盘中/午休/收盘/休市** 五状态
- **精度标注**：所有盘中估算值标注「盘中估算」
- **手动触发**：不设 cron——用户说「看一下盘面」时才跑
- **微信推送**：简报通过 `send_message` 推送到微信

## 分钟线结构 CSV 列名

`minute_structure_v2.py` 输出的 CSV 列名：

| 列 | 含义 |
|------|------|
| `top_form_60/90/120` | 顶结构形成标志 |
| `bottom_form_60/90/120` | 底结构形成标志 |
| `bd_60/90/120` | 底钝化标志 |
| `td_60/90/120` | 顶钝化标志 |
| `signal_level` | 综合评级（S/A/B/C/无结构） |
| `top_resonance` | 顶结构三周期共振计数 |
| `bot_resonance` | 底结构三周期共振计数 |
| `golden_cross_res` | 金叉三周期共振计数 |
| `dead_cross_res` | 死叉三周期共振计数 |

**坑**：不要假设列名叫 `60min_signal` 或 `resonance_level`——这些不存在。实际列名见上表。
