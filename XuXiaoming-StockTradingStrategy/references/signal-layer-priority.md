# 三层信号时序关系 + 四时段完整推演

## 核心规律

```
共振松动 → 结构预警 → 通道确认
(L2最早)    (L3中期)    (L1最后)
```

三层信号有固定的时序先后关系。逐日引擎模拟证明：通过跨日记忆（记住最近一次结构形成日期），可以在无前向偏差的条件下捕获信号。

## 四时段推演

### 2021-02 3731大顶
- 2/10: 共振 5↑0↓ + 顶钝 = 持股警戒
- 2/18: 共振 4↑0↓（松动）+ 顶钝 = 高度警惕
- 2/22: 转变期 + 顶成 = 退出
- 2/26: 单边_下跌 + 4↓ = 空仓
- **共振从 5→4→3 逐级衰减，比通道转向前提了两周**

### 2024-02 2635大底
- 1/18: 底成 + 全空共振 = 空仓警戒（结构形成但趋势压制）
- 1/30: 共振从 6↓→4↓（松动）= 空仓等
- 2/5: 共振回 6↓（最后恐慌）= 空仓等
- 2/27: 通道突破 + 偏多共振 = 买入
- **结构 1/18 形成后共振从 6→4→震荡，共振在中段松动**

### 2024-09 924突破（教科书级）
- 9/18: 全空 + 底钝 = 空仓警戒
- 9/19: 全空 + 底成 = 空仓等
- 9/24: 通道突破 + 混合(3↑)= **买入**
- 9/27: 全多 = 持股
- **底成→通道突破→全共振，3个交易日三层全翻**

### 2020-03 疫情底（纯V型）
- 无任何结构信号
- 纯共振驱动 V 型反转
- 3/19 全空 → 4/7 震荡
- **通道系统对 V 型底结构性地滞后——这是方法论边界**

## 时序规则

```
if 共振松动 and 结构形成:
    → 准备操作（降置信度但不行动）
elif 结构形成 and 通道未突破:
    → 空仓等/持股警戒（趋势压制结构）
elif 通道突破 and 近期有结构形成:
    → 执行操作（如 924: 突破+5天前底成）
elif 通道突破 and 无结构信号:
    → 纯趋势驱动，按正常规则执行
```
