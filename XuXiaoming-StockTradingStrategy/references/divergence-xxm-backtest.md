## 选型结论

| 层级 | 选用方法 | 原因 |
|------|------|------|
| 日线 | 极值法 20d | 假信号5-15%，灵敏度优于40d，钝化不入裁决 |
| 分钟线 | 极值法 20d | 与日线统一口径 |
| 顶钝化 | 单独标注 | 90min转化率93-98%，单个信号即可靠 |
| 底钝化 | 标注⚠ | 分钟线假信号46-100%，需结合日线信号判断 |
| 钝化角色 | 纯分析层 | 不入裁决规则，不影响verdict_main/tech |

## 7/13 三种方法实战对比

| 方法 | 日线底钝化 | 分钟线底钝化 |
|------|:--:|:--:|
| 极值40d | ✗ 全线盲 | — |
| 极值20d | ✗ 全线盲 | ✓ 上证/深证/创业板三周期 |
| XXM柱状线 | ✓ 三指数 | ✓ 三指数三周期 |
| 徐小明博客 | ✓ 三周期 | ✓ 三周期 |

极值法日线 7/13 全线盲的根因：DIF 与价格同步暴跌（DIF 也创20日新低），不符合"价格新低+DIF未新低"条件。但分钟线K线密度高，DIF 滞后于价格，极值法能抓到。
