# 裁决引擎 v6 年度一致性比对 — 2020-2022 (每年代30篇)

> 2026-07-12 · 逐篇LLM提取徐小明操作方向 × v6同日verdict

## 方法

- 每年随机抽30篇收盘后操作策略文章（`sampled_articles_v5.csv`）
- LLM逐篇阅读原文提取操作方向(持股/空仓/观望/试探)
- 与裁决引擎v6同日verdict比对
- 一致 = 方向相同(持股vs持股/持股警戒)；不一致 = 方向相反；部分一致 = 方向不矛盾但强度不同

## 年度结果

| 年份 | 一致 | 部分一致 | 不一致 | 一致率 | 市场环境 |

| **合计** | **55** | **3** | **32** | **61.1%** | |

加权一致率(部分一致计0.5): **62.8%**

## v5→v6变化

- v5 210篇一致率64.7%(加权73.4%) → v6 90篇(仅2020-2022)一致率61.1%(加权62.8%)
- 主要差异: v5含2023-2026(熊→震荡→牛市,趋势明确年份拉高整体一致率)
- 2022年: v5=90% vs v6=70% — v6在多个节点将"观望"降级为"空仓"(更激进)
- 2021年: v5=16.7%("一致")+73.3%("部分一致") vs v6=53.3%("一致") — v6减少了模糊判断

## 关键结论

1. **v6在趋势明确年份(2022)可靠（70%）**，单边熊市中规则驱动与原文高度一致
2. **震荡年(2020-2021)一致率仅50-60%**，主因v6判震荡→观望过于保守，徐在震荡中保持方向
3. **2021年指数分歧是最大噪音源**：v6用深证判偏空/空仓，徐用上证/中证500判持股，方向相反
4. **61-63%可能是v6的可验证上限**：剩余~35%来自徐的预测驱动操作、指数选择差异、结构vs趋势张力，不是参数调优能解决的
5. **v5部分一致率高但模糊**，v6更果断(空仓/持股二元化)但一致率略降——这是精确性vs覆盖面的权衡
