## v17 方法（v4.6.1 引擎 · 2026-06-25）

### 流程（v4.6.1 实际使用）

```
Step1: blind_sample.py 抽样 + 匹配本地文章
  → 60天/年 × 7年 = 420天, 本地2124篇文章匹配~300-350篇
  → 输出 /tmp/blind_test_batch.json（含全文+引擎裁决对照）

Step2: blind_split.py 拆分为20篇/批
  → 17批 × 20篇 = 332-350篇
  → 输出 /tmp/blind_batch_{NN}_full.json

Step3: 子Agent并行 LLM 提取（每批1个子Agent, 3批并发）
  → 每子Agent 1)读batch_full.json 2)逐篇语义理解 3)提取5字段 4)写_extracted.json
  → 提取prompt必须包含：否定句式防误判/时序区分/不足→不明或无从

Step4: blind_compare.py 汇总比对
  → 收集所有 _extracted.json → 逐项比对引擎裁决 → 统计一致率
```

### LLM 提取的字段定义

| 字段 | 含义 | 取值 | 判据 |
|------|------|------|------|
| 不重要 | 徐小明是否说今天不重要 | true/false | "今天不重要""过程中""不改变操作结果"→true; 有明确操作指令→false |
| 市况 | 市场大环境定性 | 上升/下跌/震荡/不明 | 原文明确描述的趋势方向 |
| 操作 | 当天/明天要执行的动作 | 减仓/加仓/观望/无 | 仅提取明确的操作建议词 |
| 结构 | 是否提到结构形成/钝化消失 | true/false | "结构形成""钝化消失"→true; "钝化还在""没形成"→false |
| 核心判断 | 一句话总结 | 自由文本 | 徐小明当天的核心判断 |

### 关键注意事项（写入子Agent prompt）

- **否定句式**："不建议追高"≠加仓；"没有结构"≠结构形成
- **时序区分**：过去完成的操作 ≠ 今天的操作建议
- **层级区分**：讨论分钟线结构 ≠ 日线结构形成
- **文件读取**：batch JSON 的 body 可能被清洗截断，需从 `/home/admin/file-transfer/徐小明/徐小明文集/` 直接读原始文件

### 比对维度

| 维度 | LLM字段 | 引擎字段 | 匹配规则 |
|------|---------|---------|---------|
| 不重要 | 不重要 | r["important"] | 双方一致：同判"不重要"或同判"重要" |
| 市况 | 市况 | l3["condition"] | 上升↔单边上升, 下跌↔单边下跌, 震荡↔震荡 |
| 操作 | 操作 | r["action"] | 减仓↔空仓/减仓/减仓防守/纠错; 加仓↔满仓/加仓/左侧试仓; 观望↔观望/结构操作 |
| 结构 | 结构 | l3["struct_form"] | 直接比对 |

### 批次处理

332篇文章分17批（每批20篇），3个子Agent并行处理，6轮完成。

实际脚本：
- `scripts/blind_sample.py` — 抽样 + 文章匹配 + 引擎对照注入
- `scripts/blind_split.py` — 拆分为子Agent可处理的批次JSON
- `scripts/blind_compare.py` — 汇总提取结果 + 引擎比对 + 一致性统计

### A10 修正说明

v16把"引擎空仓/观望 vs 徐小明多看少动"判为不一致——但两者方向相同（都不看多），仅是强度不同。v17合并为方向级：看多(持股+试探+加仓+持有) vs 非看多(空仓+减仓+观望+无)。仅跨边界反转算矛盾。A10修正使一致率从~45%提升至66.5%（+22%）。

### v17 核心教训

## 核心教训

1. **正则匹配徐小明文本不可靠** — 他的语言太精细（否定句、条件句、回顾性陈述、层级混淆）
2. **LLM 提取是正确路径** — 55% 一致率虽不完美，但反映的是真实的语义分歧而非工具噪声
3. **午评必须剔除** — 信息不足（600-800字），不足以做精确比对
4. **盲测66.5%是下限，报告视角70-75%是上限** — 盲测将完整时序叙事（趋势→震荡→来路→去路）切成独立快照，丢失仓位继承和逻辑链条上下文（v17 发现）
5. **A10修正必不可少** — 方向级比对（看多 vs 非看多）避免把强度差异误判为方向矛盾
6. **CHOP瞬时变化不可靠** — 94%触及38.2后弹回假收敛；持续<38.2≥3天才是真趋势涌现信号（v4.6.1 发现）

## 全版本核心教训

1. **正则匹配徐小明文本不可靠** — 否定句、条件句、回顾性陈述、层级混淆
2. **LLM 提取是正确路径** — 55-70% 一致率反映真实语义分歧，非工具噪声
3. **午评必须剔除** — 信息不足（600-800字），不足以做精确比对
4. **强度差异≠方向矛盾（A10修正）** — 合并看多 vs 非看多后仍有~33%真实方向反转
5. **盲测天然有下限** — 快照比对丢失时序上下文；报告视角（完整叙事弧线）可感知的一致性高于盲测数字。66.5%是下限，~70-75%是合理上限估计
6. **引擎无仓位记忆是系统性盲区** — 每日独立裁决 vs 累积仓位状态，震荡来路判断已在v4.6.1入代码，仓位继承未实现
