# 裁决引擎 v6 · CHOP 趋势可信度框架

> 2026-07-12 建成。用户纠正：CHOP 不单纯是趋势强度工具——高 CHOP 说明趋势模糊/不可靠，此时结构的作用必须突出。

## CHOP 角色定义

| CHOP | 含义 | 趋势可信度 | 结构/序列权重 |
|------|------|:--:|------|
| **< 38.2** | 方向清晰、趋势可靠 | **高** | 结构只修边，不改方向 |
| **38~62** | 方向模糊、趋势减速 | **中** | 结构权重上升，顶结构→减仓 |
| **> 61.8** | 衰竭/混乱、趋势不可靠 | **低** | **结构主导，趋势退让** |

核心逻辑：**CHOP 越高 → 趋势越不可靠 → 结构说了算。**

## 三级裁决规则

### 趋势清晰 (CHOP < 38.2): 趋势为王
- 上行/偏多 → 持股（顶结构/高9 → 仅警戒，不改动作）
- 下行/偏空 → 空仓（底结构 → 试探保留恐慌底/磨底标签）
- 震荡+清晰CHOP矛盾 → 用 bullish 方向降级为偏多/偏空（46天修正）

### 趋势模糊 (CHOP 38-62): 结构修边
- 上行/偏多 + 顶结构 → **减仓**（2022-07-05、2026-05-14 对得上）
- 上行/偏多 + 高9 → 持股(警戒)
- 下行/偏空 + 底结构 → 试探(磨底/恐慌底) + 月周低9升档
- 震荡 + 底结构 → 试探（自动标签：恐慌底/磨底/衰竭底）

### 趋势混乱 (CHOP > 61.8): 结构主导
- 顶结构 → 空仓
- 底结构 → 试探（衰竭底标签，CHOP>62）
- 高9 → 减仓；低9 → 试探
- 无信号 → 观望(偏多/偏空)

## 新增功能

### 日线九转接入
- 高9：趋势清晰→持股(警戒)，趋势模糊+上行→持股(警戒)，混乱→减仓
- 低9：趋势清晰→空仓(关注)，趋势模糊+下行→空仓(关注)，混乱→试探
- 序列参与率：5%（v5: 0%）

### 底部周期自动检出 (bottom_period)
- 20个交易日内 ≥3次同模式底结构 → 标记为"磨底期#N"/"恐慌底期#N"
- 2024-07-01至09-24：磨底期#12，42天连续标记
- 输出列 `bottom_period`：空/磨底期#N/恐慌底期#N/衰竭底期#N

### ts_recent 窗口
- 顶部结构: 5天（结构形成后持续警戒/减仓）
- 底部结构: 3天
