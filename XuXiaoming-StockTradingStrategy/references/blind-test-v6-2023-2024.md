# 裁决引擎 v6 年度一致性比对 — 2023-2024 (每年30篇)

> 2026-07-12 · 正则提取徐小明操作方向 × v6同日verdict

## 方法

- 每年随机抽30篇收盘后操作策略文章（seed=42，与v5一致）
- **正则提取**徐小明核心操作方向（看多/看空/观望），含结构/趋势辅助信号
- 与裁决引擎 v6 同日 verdict 比对（`verdict_v6.csv`）
- 一致 = 方向相同；部分一致 = 方向不矛盾；不一致 = 方向相反
- 加权一致率 = (一致 + 部分一致×0.5) / 可比对项

| 2023 | 5 | 8 | 0 | 13 | **69.2%** | 熊→震荡 |

## 唯一不一致：2024-10-28

| 项目 | 内容 |
|------|------|
| 徐操作 | 减仓/防守（看空）——正则捕获"不建议追涨" |
| 引擎 | 持股（看多）——趋势模糊向上 CHOP=50 |
| 实际语义 | 徐原文："不建议追涨……但从持股角度没有卖出理由"→实际更偏**观望** |
| 根因 | 正则否定语境误判。若判为观望则变为"部分一致" |

| 类型 | 篇数 | 说明 |
|------|:--:|------|
| ④ 徐观望·引擎看空（引擎更警惕）| 5 | 引擎在偏空/下行趋势中判空仓，徐表达观望 |
| ③ 徐观望·引擎看多（引擎更积极）| 2 | 引擎在偏多趋势中判持股，徐表达观望 |

**核心模式**：时期三徐小明表达更简练，"观望/等待"类表达显著增加（建议不变、耐心等）。引擎在趋势模糊时比徐更积极——判方向而非观望。

## 正则提取局限性（时期三特有问题）

时期三（2024-2026）徐小明表达式极简，"建议不变""耐心等""随缘"为主要操作语言。正则提取面临：

1. **否定语境误判**："没有卖出的理由""不建议追涨"中"卖出""追涨"被正则捕获为操作指令
2. **描述性文本误判**："各路抄底资金"中"抄底"是描述市场行为，非操作建议
3. **隐含信号缺失**："持股就好""趋势之上持股"等积极持有信号需宽泛模式匹配
4. **提取率低**：仅33-43%文章能提取操作方向（vs v5 LLM可提取87-100%）

**推荐**：做严肃一致性比对时使用LLM提取（`llm-extraction-method.md`）。正则仅适用于快速初筛。

## 关键结论

1. **2024年v6首超v5**：75.0% vs 68.5%，v6的磨底/序列升档/CHOP框架在转折年份有效
2. **零方向矛盾**：2023年0不一致，2024年仅1条（正则误判），方向一致性远高于2020-2022（61.1%）
3. **时期三的"观望化"是语言特征而非方法分歧**：徐在2023-2024表达更谨慎，但底层方向与引擎高度一致
4. **震荡市引擎表现稳健**：12条震荡市案例中6致+6部分，加权75%，零方向矛盾
