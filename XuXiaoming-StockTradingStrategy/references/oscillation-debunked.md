# 震荡市本质：不重要时期 · 数据验证

> 2026-07-11 裁决框架简化讨论中得出。三项关键发现推翻原 v5 引擎设计。

## 发现1：震荡仅 15% 交易日

| 市况 | 天数 | 占比 |
|------|:--:|:--:|
| 单边_上升 | 681 | 37.4% |
| 单边_下跌 | 583 | 32.0% |
| 震荡类合计 | 277 | **15.2%** |
| 中间态（转变期+收敛末期） | 222 | 12.2% |

50 个震荡段，中位持续 6 天，最长 32 天——从未超一个月。60% 的震荡段在 7 天以内结束。

**结论**：震荡不是"需要精细工具捕捉的复杂阶段"，是短暂的过渡。徐小明"多看少动"——就是让你等。

## 发现2：⚡立即确认制造 60 次噪音

震荡→单边切换共 91 次，其中：

| 前状态 | 次数 | 假突破率 | 中位持续 |
|------|:--:|:--:|:--:|
| 震荡_收敛 | 6 | 16% | 12天 |
| 收敛末期 | 8 | 37% | 7天 |
| **转变期⚡** | **60** | **38%** | **5天** |
| 震荡_发散 | 17 | 17% | 16天 |

转变期⚡贡献了 66% 的切换信号，但中位持续才 5 天、38% 是假突破——信号发得太早太频繁。

**结论**：⚡立即确认应废止。震荡中不做判断，等通道 5 天确认。

## 发现3：L1 过滤让假突破率翻倍

| 过滤条件 | 次数 | 假突破率 |
|------|:--:|:--:|
| 无过滤 | 91 | 32% |
| L1 同向（任一） | 15 | **60%** ⬆️ |
| L1 逆向 | 35 | 28% ⬇️ |

L1 同向信号的假突破率反而翻倍——月周九转是滞后指标，高9之后继续涨、低9之后继续跌是常态。追进去刚好撞回调。

**结论**：L1（月周九转）的正确用法是定大方向基调，不参与日线信号加减分。

## 简化后的框架

```
L1 月周九转 → 定方向基调（做多/做空/中性），不参与日线裁决
L2 日线通道 → 收盘价突破/破位 + 5天持续确认 = 单边
                未突破/破位 = 观望（震荡）
    L2a 单边 → 结构修边（顶钝/底钝→顶成/底成，不改仓位）
    L2b 震荡 → 等。不需要连错、不需要收敛检测、不需要⚡
L3 辅助 → 序列/浪型加分
L4 分钟线 → 接口预留
```

## 发现4：通道分类器86%"单边"是伪趋势

通道分类器（MA20/MA60）标了 143 段"单边趋势"，但按实际涨跌幅验证：

| | 上升"趋势" | 下跌"趋势" |
|------|:--:|:--:|
| 段数 | 79 | 64 |
| 中位涨跌 | 0.6% | 0.7% |
| 涨跌<2% | 75% | 69% |
| 持续≤3天 | 43% | 47% |
| **伪趋势率** | **87%** | **84%** |

伪趋势 = 涨跌幅<5% 或 持续<5天。86% 的"单边"根本不构成趋势。

**重算"不重要"时间**：
```
伪趋势（判了单边但没涨/没跌）：38.2%
震荡                            ：15.2%
中间态（转变期/收敛末期）         ：12.2%
─────────────────────────────────────
不重要合计                       ：65.6%  ← 徐小明"大部分时间不重要"
真正单边                         ：34.4%
```

通道分类器把三分之二的时间分错了——伪趋势标成单边，中间态造了假突破。**MA20/MA60 通道参数不是徐小明原文公开的"趋势量化标准"——具体参数在收费课程中。**

淘汰项：
- ✗ ⚡立即确认（转变期/收敛末期/局部收敛当日突破）
- ✗ 通道宽度<1.5%阈值
- ✗ 收敛≥4天判定
- ✗ 收敛末期≥10天标注
- ✗ L1 同向/逆向加减分
- ✗ 震荡→单边转换信号
- ✗ MA20/MA60 作为趋势量化标准（86%伪趋势——徐小明实际参数未公开）
