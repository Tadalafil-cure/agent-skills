## 一、总纲：三层优先级体系

### 1.1 核心铁律（8年从未动摇）

```
趋势为王 → 结构修边 → 序列辅助
```

**不可颠倒的三级体系**。趋势定方向，结构定操作时机，序列仅作辅助参考。

| 层级 | 角色 | 可自动化 | 稳定性 |
|------|------|:--------:|:------:|
| **趋势** | 定大方向、多空基调 | 是 | ★★★★★ 稳定 |
| **结构** | 定买卖点、操作时机 | 是 | ★★★★★ 稳定 |
| **序列** | 辅助确认、精确定点 | 是 | ★★★★★ 稳定 |

> **关键原文引述**（20190103）：「趋势为王，结构修边，序列辅助——三层优先级不可颠倒。」

### 1.2 核心逻辑链

```
绿角线/红角线 → 钝化 → 两种结果：
  ├── 结构形成（DIF穿DEA）→ 对应24倍反向运行周期 → 结构引发趋势改变 → 趋势接管
  └── 钝化消失（DIF创新高/低）→ 短期（约2周）无新结构 → 纠错标准
```

> **2020年核心贡献**：「钝化消失 vs 结构消失的严格区分——结构消失可能有二次结构，钝化消失则短期内再无结构。以钝化消失为纠错标准，非结构消失。」（line 2418）

## 二、周期层次体系

### 2.1 四大标准周期

| 周期 | 日K线数 | 收盘判定时点 | 对应调整周期 | 用途定位 | 可自动化 | 稳定性 |
|------|:------:|-------------|:-----------:|---------|:--------:|:------:|
| **日线** | 1根 | 15:00 | 24个交易日 | 趋势+结构基准，大周期定方向 | 是 | ★★★★★ 稳定 |
| **120分钟** | 2根 | 11:30 / 15:00 | 约12个交易日 | 浪型定位核心，全年级别结构 | 是 | ★★★★★ 稳定 |
| **90分钟** | 3根 | 11:00 / 14:00 / 15:00 | 约8个交易日 | 中型结构+序列，精确到分钟 | 是 | ★★★★ 强化 |
| **60分钟** | 4根 | 每小时整点 | 约6个交易日 | 短线买卖点，精确定时 | 是 | ★★★★ 稳定 |

> **周期与交易日换算公式（2020年确立）**：
> - 结构对应反向运行周期 = 24倍周期的K线数
> - 60分钟结构：24×60分 = 6个交易日（4根/天×6天=24根）
> - 90分钟结构：24×90分 → 约8个交易日（3根/天×8天≈24根，分钟线不跨天取8天）
> - 120分钟结构：24×120分 → 约12个交易日（2根/天×12天=24根）
> - 日线结构：24个交易日

> **关键原文引述**（20250527）：「90分钟结构的调整周期是8个交易日，不是9个。因为分钟线不跨天计算，一天240分钟，90分钟一天3根K线，24根K线对应8天，而非9天。」

### 2.2 辅助周期（降级使用，不可独立操作）

| 周期 | 日K线数 | 地位 | 使用条件 | 可自动化 | 稳定性 |
|------|:------:|------|---------|:--------:|:------:|
| **30分钟** | 8根 | 精细辅助 | 仅当60分钟以上无信号时，用于钝化+序列左侧定点 | 是 | ★★★ 退化 |
| **15分钟及以下** | 16根+ | 基本不用 | "周期级别太小，无法完成对趋势的改变" | 否 | ★★★ 退化 |

> **关键原文引述**（20220127）：「为什么不看60分钟以下的周期，最主要的原因就是周期级别太小，无法完成对趋势的改变。」

### 2.3 周期选择的动态场景化规则

**核心原则**：哪个周期有钝化/结构，就聚焦哪个周期。非僵化固定。

| 市场环境 | 核心周期 | 原因 | 稳定性 |
|---------|---------|------|:------:|
| 趋势行情 | 60分钟 | 结构可靠性高，短线精准 | ★★★ |
| 震荡行情 | 90/120分钟 | 60分钟结构频繁消失，不够稳定 | ★★★★ |
| 极端行情 | 日线+全周期 | 加速行情中钝化容易消失，需大周期结构 | ★★★★ |
| 慢牛环境 | 90/120分钟 | 分钟线仅战术风控，大方向看多 | ★★★★ 强化 |

> **审查发现**：2019年称60分钟为"核心操作周期"，2021年降级为"不稳定"，2023年起90/120分钟成为主流——这并非方法论矛盾，而是市场环境驱动的适应性调整。

### 2.4 周期K线时间切片规则

| 周期 | K线时间分布 | 收盘判定时间点 |
|------|-----------|--------------|
| **60分钟** | 10:30 / 11:30 / 14:00 / 15:00 | 每根K线结束时点 |
| **90分钟** | 11:00 / 14:00 / 15:00 | 11:00、14:00、15:00 |
| **120分钟** | 11:30 / 15:00 | 11:30、15:00 |
| **日线** | 全天一根 | 15:00 |

> **关键规则**：结构形成以**该周期收盘价**为准，盘中形成不算。120分钟有两个收盘时点（11:30和15:00）。

## 三、钝化判定规则

### 3.1 钝化定义与必要条件

**钝化**：价格创新高/新低，但MACD的DIF值未同步创新高/新低的状态。

| 条件 | 顶部钝化 | 底部钝化 | 可自动化 |
|------|---------|---------|:--------:|
| 价格条件 | 价格 > 前N个周期最高价 | 价格 < 前N个周期最低价 | 是 |
| DIF条件 | DIF值 < 前DIF峰值 | DIF值 > 前DIF谷值 | 是 |
| 前置条件 | 有红角线（MACD柱>0） | 有绿角线（MACD柱<0） | 是 |

> **关键原文引述**（2019-2026 全确认）：「钝化是结构形成的必经阶段。没有钝化肯定没有结构。」

### 3.2 钝化级别判定因素

| 因素 | 级别大（结构形成概率高） | 级别小（钝化消失概率高） | 可自动化 | 稳定性 |
|------|----------------------|----------------------|:--------:|:------:|
| **DIF值差距** | DIF距前峰值远 | DIF接近前峰值 | 是 | ★★★★★ 稳定 |
| **角线数量** | 红/绿角线数量多（≥3天反弹） | 角线少或无 | 是 | ★★★★★ 稳定 |
| **钝化周期数** | 钝化已维持多个周期 | 刚出现钝化或速度快 | 是 | ★★★★ 稳定 |
| **速度** | 缓速运行（衰退期） | 加速运行（鼎盛期） | 是 | ★★★★★ 稳定 |
| **横盘DIF下移** | 横盘中DIF值持续下移→级别增大 | 单边行情中DIF走平 | 是 | ★★★ 强化 |
| **多周期共振** | 多周期同时钝化 | 单周期钝化 | 是 | ★★★★★ 稳定 |
| **多指数共振** | 多指数同时钝化 | 单指数钝化 | 是 | ★★★★ 强化 |

> **关键原文引述**（20220209）：「反弹需要至少三个交易日，才能在90分钟和120分钟形成较大的红角线，红角线越多，将来钝化的级别越大。」

### 3.3 钝化维持周期估算

```
钝化可维持周期数 ≈ (当前DIF值 - 前DIF峰值) / DIF日均变化速度
```

> **可自动化**：是 | **稳定性**：★★★★ 强化（2025-2026精确到个位数点位）

### 3.4 钝化消失阈值精确计算

逐年精确化演进：
- 2020年：基于DIF值/峰值比例估算
- 2025年：精确量化点位（如3404点、3335点）
- **2026年**（极致）：「上涨6点钝化消失，下跌16点结构形成」——精确到个位数点位

> **可自动化**：是 | **稳定性**：★★★★★ 强化

### 3.5 反弹→红角线→钝化因果链（2022年系统化）

```
急跌(鼎盛) → 反弹(过渡，产生红角线) → 缓跌(衰退) → 钝化形成 → 结构形成(终结)
```

**反弹≥3天是钝化形成级別够大的必要条件。**

> **可自动化**：是（反弹检测+红角线计数+钝化检测） | **稳定性**：★★★★ 强化

## 四、结构形成规则

### 4.1 结构形成判定

**结构形成**：钝化后DIF值反向穿越DEA线。

| 判定条件 | 顶部结构 | 底部结构 | 可自动化 | 稳定性 |
|---------|---------|---------|:--------:|:------:|
| 前提 | 先有顶部钝化 | 先有底部钝化 | 是 | ★★★★★ |
| 触发条件 | DIF下穿DEA | DIF上穿DEA | 是 | ★★★★★ |
| 收盘确认 | 以该周期收盘价DIF/DEA为准 | 同左 | 是 | ★★★★★ |
| 盘中形成 | **不算**（等收盘确认） | 同左 | 是 | ★★★★★ |

> **关键原文引述**（20230627）：「结构形成要以分钟的收盘价为准。比方说90分钟，分别是11点，下午两点和下午三点。只要在这三个时间点形成，才是结构形成。不以盘中为准，以周期收盘为准。」

### 4.2 结构形成后的关键规则

| 规则 | 内容 | 可自动化 | 稳定性 |
|------|------|:--------:|:------:|
| **结构生命期** | 结构形成→对应24倍反向运行周期 | 是 | ★★★★★ 稳定 |
| **结构消失 vs 钝化消失** | 结构消失可能产生二次结构；钝化消失则无 | 是 | ★★★★★ 稳定 |
| **纠错标准** | 以**钝化消失**为纠错，非结构消失 | 是 | ★★★★★ 稳定 |
| **结构→趋势** | 结构引发趋势改变后使命完成，趋势接管 | 部分是 | ★★★★★ 稳定 |
| **收盘价至上** | 盘中形成≠结构形成，等周期收盘确认 | 是 | ★★★★★ 强化 |
| **二次结构** | 同一周期再次形成结构，准确率高于首次 | 是 | ★★★★ 强化 |
| **二次结构不重复操作** | 同一级别结构已有防守则不再操作 | 是 | ★★★★ 强化 |
| **低点≠买点** | 低点是序列判定，买点需带结构（2022年贡献） | 部分是 | ★★★★ 稳定 |

> **关键原文引述**（20220105）：「钝化消失之后，代表了短期市场不会再有钝化，没有钝化就没有结构。」

> **关键原文引述**（202008）：「以钝化消失为纠错标准——结构消失可能产生二次结构，钝化消失则短期内无新结构。」

### 4.3 价格到趋势的距离——结构可操作性判定

| 距离 | 所需结构级别 | 操作意义 |
|------|------------|---------|
| **近**（紧贴趋势线） | 小周期结构（60分钟）即可 | 高可操作性 |
| **中** | 需90/120分钟结构 | 中可操作性 |
| **远** | 需日线结构 | 低可操作性（小周期结构无用） |

> **可自动化**：是（计算价-趋势线距离） | **稳定性**：★★★★★ 稳定

> **关键原文引述**（20220124）：「如果趋势远离，小周期结构就没用了。」

### 4.4 浪型阶段与结构意义（2024年新增）

| 浪型阶段 | 分钟线策略 | 结构意义 |
|---------|-----------|---------|
| 1浪 | 不追高 | 结构仅参考 |
| 2浪A | "随缘"——不刻意找买点 | 等右侧确认 |
| 2浪B | "精确"——分钟线结构+序列定点 | 高价值 |
| 2浪C | "激进"——带结构即可操作 | 最具操作性 |
| 3浪（主升） | 持股权重，分钟线仅风控 | 不做空 |
| 3浪3 | **不容有失**（急涨不做空的极端形式） | 顶部结构也不做空 |

> **可自动化**：部分是（需浪型标注） | **稳定性**：★★★ 强化

## 五、多周期嵌套规则

### 5.1 核心嵌套原则

| 原则 | 内容 | 可自动化 | 稳定性 |
|------|------|:--------:|:------:|
| **大周期定方向，小周期定时点** | 日线定日 → 120分钟定时段 → 90/60分钟定小时 | 是 | ★★★★★ |
| **同向共振增强确定性** | 多周期同向信号→级别增大 | 是 | ★★★★★ |
| **反向矛盾→震荡** | 大小周期信号反向→市场震荡 | 是 | ★★★★★ |
| **多周期结构取大周期** | 三个周期都形成结构→以较大周期为准 | 是 | ★★★★★ |
| **从小周期逐级确认** | 先看60/90→再确认120/日线 | 是 | ★★★★ |

### 5.2 全周期共振（最强信号）

```
60分钟 + 90分钟 + 120分钟 + 日线 同时钝化/结构 → 最强信号（S级）
```

**案例**：2019年1月4日底部——四重周期共振买入（年度最佳案例）

> **可自动化**：是（四周期信号矩阵取AND） | **稳定性**：★★★★★ 稳定

### 5.3 指数协调性判定

| 协调性 | 行情级别 | 对分钟线信号影响 |
|--------|:------:|----------------|
| **协调性好**（多指数同向信号） | 大级别 | 分钟线信号可靠性高 |
| **分歧大**（指数不同步） | 震荡/小级别 | 分钟线信号打折扣 |
| **严重分化**（个股vs指数不一致） | 降低频率 | 分钟线操作价值降低 |

> **可自动化**：是（多指数信号取交集/评分） | **稳定性**：★★★★ 强化

> **关键原文引述**（2021-2026强化）：「指数之间的协调性越好，行情的级别就越大；指数之间的分歧越大，行情越倾向于震荡。」

### 5.4 标的指数统一原则

> **核心规则**：以上证指数为统一标的，保证操作唯一性。深成指/创业板作为辅助参考。

> **可自动化**：是（配置参数指定主指数） | **稳定性**：★★★★ 稳定

## 六、序列使用规则

### 6.1 序列基本定位

> **铁律**：「序列不是交易规则，仅作辅助参考。」

| 规则 | 内容 | 可自动化 | 稳定性 |
|------|------|:--------:|:------:|
| **序列≠交易规则** | 有结构没序列可以操作，只有序列不够 | 是 | ★★★★★ |
| **钝化+9 > 单独钝化 > 单独序列** | 信号强度排序 | 是 | ★★★★★ |
| **钝化+9必须同期** | 钝化和序列9必须在同一周期同时出现 | 是 | ★★★★ |
| **日线序列稀缺** | 一年约2次日线序列 | 是 | ★★★★★ |

### 6.2 九转序列计数规则

| 项目 | 规则 | 可自动化 |
|------|------|:--------:|
| **计数方式** | 连续K线同向收高/低，计数1→9 | 是 |
| **有效性条件** | 低9在阴线低位→效果好；高9在阳线高位→效果好 | 是 |
| **提前/滞后** | 低10/高10仍可能有效（±1周期容差） | 是 |
| **序列重排** | 价格反向运行→序列被打断→重新计数 | 是 |
| **序列消失** | 跳空可导致序列消失→原判断不成立 | 是 |

### 6.3 序列评分体系（已退化，简化为二元）

| 时期 | 评分方式 | 结论 |
|------|---------|------|
| 2019-2020 | 多维度评分（波段/趋势/历史波动率/共振等） | 120分钟最高分98分 |
| 2021-2026 | 简化为"高分"/"低分"二元 | 评分体系退化 |

> **退化原因**：不同市场阶段评分变化大，不易标准化。

### 6.4 波动率不相邻规则（2020年独有，后续未使用）

> 序列波动率只能在一两个非相邻周期有效。该规则在2020年提出后未被广泛使用。

> **可自动化**：部分 | **稳定性**：★★ 退化

### 6.5 多周期序列共振

```
四周期同时出现底部序列（30+60+90+120分钟）→ 极罕见，确定性极高
```

**案例**：2019年6月4日——"四个相邻的分钟周期全部出现了底部序列，这种情况非常少见。"

> **可自动化**：是 | **稳定性**：★★★★ 稳定

## 七、一致性获利法（穿轴）

### 7.1 核心定义

调整MACD参数(a, b, c)后，当DIF上穿/下穿零轴时判定4浪结束。

| 规则 | 内容 | 可自动化 | 稳定性 |
|------|------|:--------:|:------:|
| **参数动态调整** | 参数b根据1浪起点到3浪终点K线数动态计算 | 是 | ★★★★★ |
| **穿轴判定** | DIF上穿0轴=4浪反弹高点；下穿0轴=4浪调整低点 | 是 | ★★★★★ |
| **穿轴+止跌/止涨** | 穿轴后止跌→确认4浪；不止→排除4浪 | 部分是 | ★★★★ |

## 八、速度判定规则

### 8.1 速度——决定钝化/结构命运的核心变量

| 规则 | 内容 | 可自动化 | 稳定性 |
|------|------|:--------:|:------:|
| **加速干掉一切钝化** | 2022年确立为铁律 | 是 | ★★★★★ 强化 |
| **急涨不做空 + 急跌不抄底** | 速度过快时放弃反向操作 | 是 | ★★★★★ 稳定 |
| **速度↑→钝化消失概率↑** | 速度越快，钝化级别越小 | 是 | ★★★★★ 强化 |
| **速度↓→结构形成概率↑** | 缓速运行更易形成结构 | 是 | ★★★★★ 强化 |

### 8.2 鼎盛→衰退→终结三阶段模型

```
鼎盛期（急涨/急跌，速度快，无钝化）
  ↓ 反弹/调整（过渡，产生角线）
衰退期（缓涨/缓跌，速度降，钝化形成）
  ↓ DIF穿DEA
终结（结构形成，趋势反转）
```

> **可自动化**：部分是（速度+角线+钝化组合判定） | **稳定性**：★★★★★ 强化

### 8.3 速度定量标准

| 判定 | 标准 | 性质 | 可自动化 |
|------|------|------|:--------:|
| 鼎盛（角度>60°） | 极速涨跌 | 属于下跌浪/主升浪 | 是（斜率计算） |
| 衰退（角度<30°） | 缓速涨跌 | 属于调整浪 | 是（斜率计算） |
| 角度45° | 中等速度 | 需结合其他因素 | 是 |

> **⚠️ 注意**：徐小明2019年提出的15°/45°/60°角度量化仅为比喻，非严格的几何角度。回测时可用价格变化率/ATR替代。

### 8.4 大小周期同向/反向与速度关系

| 方向关系 | 速度 | 对钝化影响 |
|---------|------|-----------|
| 大小周期同向 | 加速 | 钝化消失概率极高 |
| 大小周期反向 | 缓速/震荡 | 钝化→结构形成概率大 |

> **可自动化**：是（多周期趋势方向×速度矩阵） | **稳定性**：★★★★★ 稳定

### 8.5 速度判定小结

```
快速行情（鼎盛）→ 钝化级别小 → 钝化消失概率大 → 短期无结构 → 不操作
缓速行情（衰退）→ 钝化级别大 → 结构形成概率大 → 可操作
```

> **退化信号**：角度量化速度（15°/45°/60°，2019年特有）已退化，被DIF值定量方法替代。

## 九、消散机制

### 9.1 三大消散类型对比

| 类型 | 定义 | 触发条件 | 后续影响 | 可自动化 |
|------|------|---------|---------|:--------:|
| **钝化消失** | DIF值创新高/新低，钝化条件不满足 | 价格持续加速运行 | 短期（约2周）无新结构，纠错标准 | 是 |
| **结构消失** | 结构形成后DIF反向穿越回原区间 | 价格反向波动 | 可能出现新钝化→二次结构 | 是 |
| **序列消失/重排** | 跳空或反向运行打断序列 | 价格反向运动/跳空 | 原有序列判断失效，重新计数 | 是 |

### 9.2 钝化消失——最严格的消散

| 规则 | 内容 | 可自动化 | 稳定性 |
|------|------|:--------:|:------:|
| **钝化消失→纠错** | 以钝化消失为唯一纠错标准 | 是 | ★★★★★ |
| **消失后无结构期** | 约2周内不会再有新结构（"两周规则"，2025年明确） | 是 | ★★★★★ 强化 |
| **消失阈值推算** | DIF变化速度×剩余周期数，与峰值比较 | 是 | ★★★★ 强化 |
| **模拟K线预判** | 用上一周期涨跌幅推算下一周期，判断钝化是否消失 | 是 | ★★★★ |

### 9.3 钝化消失 vs 结构消失的严格区分（2020年核心贡献）

```
结构消失 ≠ 纠错标准
钝化消失 = 纠错标准

结构消失后：可能出现新钝化→二次结构→仍可操作
钝化消失后：短期内再无钝化→再无结构→不操作
```

> **关键原文引述**（20200820）：「结构失效的判定标准是以钝化消失为标准。钝化消失主要就是应对这种多重结构的，如果你不是以钝化消失为标准而是以结构消失为标准，遇到这种多重结构的时候，执行起来就是噩梦。」

### 9.4 错误结构（形成即消失）

| 特征 | 原因 |
|------|------|
| 结构形成后短时间内消失 | 级别小+速度快 |
| 盘中形成但收盘未确认 | 盘中不算结构形成 |
| 结构形成后DIF快速反向穿越 | 假突破/错误结构 |

> **可自动化**：是（"形成→消失"短周期内反转检测） | **稳定性**：★★★★ 稳定

## 十、信号强度等级（S/A/B/C/D）

### 10.1 等级定义

基于**周期层级 + 多周期协调性 + 钝化级别 + 速度状态 + 指数协调性**五维度综合评定。

| 等级 | 名称 | 构成条件 | 操作意义 | 历史出现频率 |
|:----:|------|---------|---------|:----------:|
| **S** | 极强信号 | 全周期（60+90+120+日线）钝化/结构共振 + 多指数共振 + 钝化级别大 + 速度衰退 | 必须操作，极大概率反转 | 数年一遇 |
| **A** | 强信号 | 2-3个大周期（90+120+日线）同向结构 + 钝化级别大 + 多指数协调 | 应操作，高确定性 | 年数次 |
| **B** | 中等信号 | 2个周期（如90+120分钟）同向结构或钝化+9组合 + 单指数或分歧小 | 可操作，需关注趋势距离 | 月数次 |
| **C** | 弱信号 | 单周期结构或单独钝化/序列 + 速度不明朗 | 轻仓或观望，配合趋势 | 频繁 |
| **D** | 噪音 | 60分钟以下周期信号、单独序列、结构消失后 | 忽略，不操作 | 极频繁 |

### 10.2 等级评分矩阵

| 维度 | S级（4分） | A级（3分） | B级（2分） | C级（1分） | D级（0分） |
|------|:--------:|:--------:|:--------:|:--------:|:--------:|
| **周期层级** | 日+120+90+60 | 日+120+90 | 120+90或日线 | 单周期≥90 | 60分钟以下 |
| **钝化级别** | 极大（DIF距峰值>5%） | 大 | 中 | 小 | 无钝化 |
| **多周期协调** | 全共振（4/4） | 强共振（3/4） | 中共振（2/4） | 弱共振（1/4） | 无共振 |
| **速度状态** | 明确衰退+缓速 | 衰退 | 不明朗 | 鼎盛或不明 | 急涨急跌 |
| **指数协调** | 全协调（3+指数） | 强协调（2指数） | 中协调（1指数） | 分歧 | 严重分化 |

**等级阈值（总分）**：S≥18 | A≥14 | B≥9 | C≥4 | D<4

> **可自动化**：是（五维度评分矩阵） | **稳定性**：★★★★ 强化

### 10.3 信号强度衰减规则

| 情况 | 降级幅度 |
|------|:------:|
| 指数严重分化 | 降1级 |
| 价格远离趋势 | 降1~2级 |
| 钝化级别小（接近消失边缘） | 降1级 |
| 速度在加速中 | 降2级 |
| 30分钟以下周期 | 直接降至D级 |

## 十三、回测引擎构建指南

### 13.1 所需数据

| 数据项 | 重要度 | 说明 |
|--------|:----:|------|
| 指数日线OHLCV数据 | 🔴 必须 | 上证指数为主，深成指/创业板为辅 |
| 60/90/120分钟K线数据 | 🔴 必须 | 需按徐小明的K线时间切片规则生成（见2.4节） |
| MACD指标（DIF/DEA/柱） | 🔴 必须 | 默认参数(12,26,9)，EMA算法需统一 |
| 30分钟K线数据 | 🟡 可选 | 用于降级辅助判定 |
| 前复权价格 | 🟡 建议 | 保证价格连续性 |

### 13.2 核心参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| MACD快速线 | 12 | EMA周期 |
| MACD慢速线 | 26 | EMA周期 |
| MACD DEA | 9 | 信号线周期 |
| DIF/DEA初始值 | 需定义（SMA或第一根K线值） | 🟡 影响回测精度 |
| 钝化判定前峰周期数 | 建议≥5~8个周期 | 🔴 需确认（未量化） |
| 结构形成确认 | 该周期收盘时判定 | DIF穿DEA发生在收盘时确认 |
| 序列重排阈值 | 反向运行幅度待定义 | 🟡 需确认 |
| 趋势线算法 | 未公开（建议用MA20或Donchian通道替代） | 🔴 最大未知项 |

### 13.3 判定逻辑伪代码

```
# ===== 模块1：数据准备 =====
for each 周期 in [60min, 90min, 120min, 日线]:
   计算 MACD(12,26,9) → DIF, DEA, HIST
   计算 前N周期DIF峰值/谷值
   计算 价格到趋势线距离

# ===== 模块2：钝化检测 =====
for each 周期:
   if (价格创新高 AND DIF < 前DIF峰值):
      顶部钝化 = True
      钝化级别 = abs(DIF - 前DIF峰值) / abs(前DIF峰值)  # 以百分比定级
   if (价格创新低 AND DIF > 前DIF谷值):
      底部钝化 = True
      钝化级别 = abs(DIF - 前DIF谷值) / abs(前DIF谷值)

# ===== 模块3：结构形成/消失检测 =====
for each 周期:
   if (有钝化 AND DIF穿DEA):
      结构形成 = True  （以该周期收盘价为准）
      结构类型 = 顶部/底部
      结构生命期 = 24 个该周期K线
   if (结构已形成 AND 钝化消失):
      钝化消失 = True  → 纠错信号
   if (结构已形成 AND DIF反向穿DEA):
      结构消失 = True  → 不一定是纠错信号

# ===== 模块4：钝化消失检测 =====
for each 周期:
   if (DIF创新高 或 DIF创新低):
      钝化消失 = True
      标记"约2周安全期"

# ===== 模块5：序列检测 =====
for each 周期:
   九转序列计数
   if (有钝化 AND 序列=9 AND 同周期):
      钝化+9信号 = True

# ===== 模块6：速度判定 =====
for each 周期:
   速度 = (当前价格 - N周期前价格) / N
   速度方向 = 加速度 vs 减速度
   判定 = 鼎盛 / 衰退 / 过渡

# ===== 模块7：多周期协调性评分 =====
同向信号数 = sum(各周期同向信号)
协调性 = 同向信号数 / 总周期数

# ===== 模块8：多指数协调性评分 =====
同向信号数 = sum(上证/深成指/创业板同向信号)
协调性 = 同向信号数 / 总指数数

# ===== 模块9：信号强度等级计算 =====
综合评分 = 周期层级分 + 钝化级别分 + 周期协调分 + 速度状态分 + 指数协调分
等级 = if 评分>=18: S
       elif 评分>=14: A
       elif 评分>=9: B
       elif 评分>=4: C
       else: D

# ===== 模块10：一致性获利法 =====
if (浪型标注=4浪阶段):
   1浪起点到3浪终点K线数 = count_kline(start, end)
   b = round(K线数 / 1.77)
   调整MACD参数(5, b, 5)
   if (DIF穿0轴):
       4浪结束信号 = True
   if (穿轴后止跌/止涨):
       确认4浪
   else:
       排除4浪 → 升级到更大周期

# ===== 模块11：消散监控 =====
监控：钝化是否消失 / 结构是否消失 / 序列是否重排
触发条件标注纠错/非纠错
```
