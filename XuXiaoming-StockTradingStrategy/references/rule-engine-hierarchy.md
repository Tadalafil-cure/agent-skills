## 铁律（裁决前必须过）

| # | 铁律 | 原文 | 引擎实现 |
|---|------|------|---------|
| 1 | 趋势为王，结构修边 | "趋势为王结构修边" (2019-02-10) | 趋势信号优先级高于结构，结构用于过滤趋势噪音 |
| 2 | 趋势 > 结构 > 序列 | "序列不是交易规则" (2019-08-13) | 序列只能增强/削弱，不能独立触发操作 |
| 3 | 浪型不是交易原则 | "浪形也不是交易原则" (2019-08-12) | 浪型仅作参考，不参与操作决策 |
| 4 | 大周期定方向，小周期定时机 | "以往都是小周期先形成钝化再到大周期" (2019-08-07) | 月→周→日→分钟，上层覆盖下层 |

## 裁决层次（从上到下，上层否决下层）

```
┌─────────────────────────────────────────────────┐
│ Layer 0: 铁律检查                                │
│   趋势信号不可被结构/序列否决                     │
│   序列不可独立触发操作                           │
├─────────────────────────────────────────────────┤
│ Layer 1: 月线（宏观方向）                        │
│   ▸ 月线趋势方向 → 定牛熊大格局                  │
│   ▸ 月线序列低9/高9 → 6年一见，大级别信号        │
│   ▸ 月线底部结构 → 历史性大底确认                 │
│   ▸ 裁决: 月线下降中，日线上升信号→仅反弹        │
├─────────────────────────────────────────────────┤
│ Layer 2: 周线（中期节奏）                        │
│   ▸ 周线趋势方向 → 中期仓位方向                  │
│   ▸ 周线MA20/MA60通道 → 趋势强度                 │
│   ▸ 周线结构 → 数周级别转折                      │
│   ▸ 裁决: 周线上升+日线调整→上升途中调整         │
├─────────────────────────────────────────────────┤
│ Layer 3: 日线（操作核心）                        │
│   ▸ 日线趋势交叉 → 主要买卖信号                  │
│   ▸ 日线结构形成/消失 → 精确买卖点               │
│   ▸ 日线序列高9/低9 → 辅助定时                   │
│   ▸ 市况分类 → 震荡中趋势信号降级                │
│   ▸ 裁决: 震荡+纯趋势交叉→噪音，过滤             │
├─────────────────────────────────────────────────┤
│ Layer 4: 分钟线（时机精修）⭐ 待接入              │
│   ▸ 多周期共振(60/90/120) → 信号级别升级/降级    │
│   ▸ 速度过滤 → 快速运行中结构不可靠               │
│   ▸ 钝化加9 → 精确时点                           │
│   ▸ 裁决: 日线有信号+分钟线共振→级别大            │
└─────────────────────────────────────────────────┘
```

## 层内裁决规则（每层内：趋势 → 结构 → 序列）

### Layer 3 日线（已实现）

| 优先级 | 规则 | 条件 | 裁决 |
|--------|------|------|------|
| 3.1 | 趋势突破/破位 | 多指数共振 + 非震荡市 | 触发操作（满仓/空仓） |
| 3.1b | 趋势交叉降级 | 震荡市 + 无结构配合 | 降为噪音，不触发 |
| 3.1c | 趋势交叉降级 | 10天≥3次交叉 | 反复穿越，不触发 |
| 3.2 | 结构形成 | 单边上升+顶部结构 | 减仓防守 |
| 3.2b | 结构形成 | 单边下跌+底部结构 | 左侧试仓 |
| 3.3 | 钝化消失 | 任何市况 | 纠错信号 |
| 3.4 | 序列增强 | 日线高9+上升趋势 | 序列让位趋势，仅警告 |
| 3.5 | 连错共振 | ≥5次+≥2指数 | 可能切换单边，警告 |

### Layer 4 分钟线（待实现）

| 优先级 | 规则 | 条件 | 裁决 |
|--------|------|------|------|
| 4.1 | 多周期共振升级 | 日线信号 + 60/90/120至少2个同步 | 信号级别升级，置信度+1 |
| 4.2 | 多周期缺失降级 | 日线信号 + 分钟线全无 | 孤立信号，级别小 |
| 4.3 | 速度过滤 | 分钟线快速单边 | 结构中信号不可靠 |
| 4.4 | 钝化加9 | 分钟线钝化+序列9共振 | 精确时点，可提前预警 |
| 4.5 | 小周期领先 | 分钟线出钝化+日线未出 | 预警：日线可能随后出 |

## 跨层裁决规则

| 优先级 | 规则 | 条件 | 裁决 |
|--------|------|------|------|
| C.1 | 月线否决 | 月线下降+日线上升信号 | 日线上涨→反弹，不追 |
| C.2 | 周线修正 | 周线上升+日线调整 | 上升途中调整，等待结束 |
| C.3 | 三周期共振 | 月+周+日同向 | 信号可信度最高 |
| C.4 | 日周冲突 | 日线方向与周线相反 | 逆大势信号，级别有限 |
