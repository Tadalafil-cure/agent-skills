## 一、徐小明对波浪理论的基本定位

### 核心矛盾：高度依赖又反复否定

徐小明是少见的用量化工具（MACD参数调整、一致性获利法）改造波浪理论的技术分析师，但他同时反复声明"浪型不是规则"。这个矛盾贯穿了他的整个职业生涯。

> "波浪理论用来描述市场所处的位置非常方便，我说1浪大家就知道是刚开始，我说3浪就知道是鼎盛，我说5浪就知道是最后一浪，快结束了。对于行情的定位，没有任何一个理论可以超过波浪理论。可是并不代表了，我们的交易原则是波浪体系的。"（2019）

> "浪型不是规则。按规则来，目前没有结构，趋势也完好，结论也是持股待涨。"（2025）

**实际使用频率反而揭示真相**：

| 年份 | "波浪理论"提及 | 时期 | 外部市况 |
|:--:|:--:|:--:|------|
| 2019 | 7 | 一 | 趋势上涨（高频使用） |
| 2020 | 2 | 一 | 疫情冲击（使用骤降） |
| 2021 | 3 | 一→二 | 震荡市（维持低频） |
| 2022 | 7 | 二 | 趋势下跌（反弹期使用） |
| 2023 | 2 | 二→三 | 底部震荡（最低） |
| 2024 | 1 | 三 | 快速反转（近乎废弃） |
| **2025** | **18** | 三 | **3浪主升（爆发式回升）** |
| 2026 | 0 | 三 | 震荡（回归废弃） |

**规律**：波浪理论的使用频率与行情结构清晰度高度相关。在明确的5浪推动结构中（2019年初、2025年3浪主升），使用频率高；在震荡市中几乎不用。他没有放弃这个工具——行情本身决定了它什么时候有用。

## 二、徐小明对传统波浪理论的改造

### 改造1：一致性获利法（时间跨度研究）

这是徐小明对波浪理论最重要的量化贡献。他将 MACD 参数按相邻波段的 K 线数调整，用 DIF 上穿零轴来精确定位 4 浪终点。

**公式**：

```
MACD 参数 = (5, 34, 5)  → 通用参数
MACD 参数 = (5, N/2+1, 5) → N = 1浪起点到3浪终点的K线数

当 DIF 在调整后参数下"上穿零轴" → 4浪结束点
```

**原文**（2019）：
> "从1浪起点到3浪终点的K线数是62根线，应用于一致性获利法的时间跨度研究，MACD参数应该调整为5、35、5，当DIF值在这个参数的条件下'上穿'零轴，则见120分钟线的4浪高点。"

**判据**：
- 如果市场在穿轴附近止跌反弹 → 确认 4 浪，将走 5 浪
- 如果价格继续下跌远超 2 浪规模 → 排除 4 浪，浪型划分重新来
- 如果穿轴后不跌反涨 → 不是 4 浪，前面全部重划

### 改造2：极值定价 vs 收盘定价

这是徐小明发现的波浪理论和自己方法论体系之间的关键差异：

> "我讲的各种各样的方法里，不论是时间周期的计算还是定量结构的计算，还是单阳测顶单阴测底，都是采用的收盘价格。一些经典的指标，MACD，KDJ，均线等都是采用收盘价格。**唯独浪型也就是波浪理论是采用的极值价格**，就是最高价或最低价。"（2021）

这意味着波浪理论的确认信号（3浪必须过1浪最高点、5浪必须低于3浪最低点）与他的"一切尽在收盘"原则存在天然的张力。他在实际操作中更倾向于用收盘价确认，但承认波浪理论的标准是极值价。

### 改造3：ABC 的排除法而非确认法

> "波浪理论标准的ABC浪，B浪的高点只要不超过A浪的起点，就都可以暂时先判断ABC的。也就是说，**它不一定能判断什么是B浪，但可以判断什么不是B浪。**"（2019）

这是徐小明对波浪理论最务实的改造——不追求正向确认浪型（因为那需要事后验证），而是用排除法：某个走法不符合任何已知浪型定义 → 排除该可能 → 剩余的可能中取概率最大者。

## 三、徐小明的浪型判断规则

### 推动浪（1-2-3-4-5）

| 浪 | 判据 | 徐小明注 |
|:--:|------|------|
| 1浪 | 趋势初始突破后的第一波 | 事后才能确认，当时看不出来 |
| 2浪 | 不低于1浪起点；常与1浪时间对称（0.5-0.618） | "2浪会让人误以为熊市没有结束" |
| 3浪 | 必须过1浪最高点（极值价） | "主升浪，速度最快、时间最长、幅度最大" |
| 4浪 | 与2浪时间规模对称；穿轴确认 | "止跌但不涨"——等待时间消耗 |
| 5浪 | 必须过3浪最高点（极值价）；速度衰退 | "衰退浪，空间小、时间长" |

**3浪确认的三阶段**（2025年明确）：

```
阶段1：突破趋势通道 → 全面看多
阶段2：站上3450点 → 摆脱震荡箱体
阶段3：突破3674点 → 波浪理论标准确认（过1浪顶）
```

> "但是等到过3674点再判断3浪肯定晚了，所以这里我们建议过趋势就要树立3浪的意识，坚决的看多做多。"（2025）

### 调整浪（ABC）

| 浪 | 判据 | 徐小明注 |
|:--:|------|------|
| A浪 | 推动浪结束后的第一波反向 | 事后确认 |
| B浪 | 不超过A浪起点 | "不一定能判断什么是B浪，但可以判断什么不是B浪" |
| C浪 | 必须低于A浪低点（极值价） | C浪与A浪在时间和空间上应有对称性 |

**ABC 确认规则**：

```
B浪高点 < A浪起点 → 暂定ABC（不可排除）
C浪低点 < A浪低点 → 正式确认C浪
C浪走完 + A浪时间对称 → 调整结束概率高
```

### 浪型排除规则

1. **排除4浪**：调整规模远超2浪 → 不是4浪 → 重新划分
2. **排除ABC**：B浪高点超过A浪起点 → 不是ABC → 可能是12345推动
3. **排除5浪**：不创新高且跌破趋势 → 可能根本没有5浪
4. **排除3浪**：价格跌破1浪起点 → 不是3浪回调 → 是新的下跌推动

## 四、浪型在交易体系中的实际位置

### 官方定位（徐小明自述）

```
趋势为王 > 结构修边 > 序列辅助 > 浪型参考
                 ↑              ↑
            交易规则        不是交易规则
```

> "在我的体系里，波浪理论并不是交易原则……交易原则是趋势为王结构修边。"（2019）

### 实际使用模式

| 场景 | 浪型怎么用 |
|------|------|
| 描述当前位置 | "目前处于3浪上升初期"——帮读者理解市场阶段 |
| 预判后市结构 | "4浪之后还有5浪"——提前布局等结构 |
| 增强结构信号 | "5浪末期+底部结构"→确定性更好 |
| 解释失败信号 | "那个结构之所以失效，因为当时在3浪主升中" |
| 不用于 | 入场/出场决策本身 |

### 与时间对称的关系

波浪理论是徐小明时间对称的**上游框架**——必须先划分浪型（这是3浪1、那是3浪2），才能谈"3浪1 ≈ 3浪2 的天数"。没有浪型标注，时间对称就失去了"同级波段"的定义。

这也就是为什么时间对称一旦脱离人工浪型划分就无法量化——因为浪型本身就无法量化。

## 五、工具层面的具体操作

### 一致性获利法（穿轴）

徐小明用 MACD 参数调整后的 DIF 上穿零轴来定位 4 浪终点，这是他少数可以量化的波浪工具。

**步骤**：
1. 确认 1 浪起点和 3 浪终点的 K 线数 N
2. MACD 参数调整为 (5, N/2+1, 5)
3. 观察 DIF 上穿零轴的时点
4. 穿轴时如出现结构 → 4 浪确认；继续下跌远超穿轴 → 排除 4 浪

**失效条件**：市场不走 5 浪结构（震荡市中经常失效）；N 的计数依赖人工判断。

### 浪型追踪

曾编写简版浪型追踪脚本（139 个高低点、时间对称检测），因波浪量化的主观性较强，常规量化难以实现，调参容易过拟合，且波浪理论不入引擎，故相关脚本已清理，代码不再保留。

## 六、量化困难

### 为什么波浪理论在当前条件下无法量化

1. **浪型确认是事后性的**："只有超过3浪的最高点才能算作是5浪上升"——等到确认时，行情已经走完了
2. **浪型划分有多个竞争版本**：同一个 K 线图，三个分析师可能划出三种不同浪型，每一种在当下都有概率优势
3. **参数依赖人工计数**：N（1浪起点到3浪终点的K线数）是手工数的，起止点选择有主观性
4. **徐小明自己的使用频率随行情波动**：2019年55次、2023年18次、2024年近乎废弃——规则不稳定
5. **极值价 vs 收盘价的张力**：方法论体系以收盘价为准，波浪理论以极值价为准，两者在同一系统中产生摩擦

### 可能的量化方向（供后续参考）

- **浪型不预判，只回溯**：不对当前浪型做实时标注，只在波段明确结束后做历史标注。回溯标注可用于训练时间对称的"同级波段"定义。
- **一致性获利法自动化**：穿轴检测是纯数学运算，可以自动化——问题在于 N 的自动计数。
- **5 浪结构检测**：用价格极值点的序列特征检测"更高高→更高高→衰竭→更低高→新低"的三推结构，这是传统技术分析中可以编码的部分。

## 七、徐小明自己怎么评价波浪理论的精准度

他不相信它能精准。原文为证：

### "5浪的概率并不超过50%"

> "首先我并不认为一定会出现5浪下跌，甚至我认为这种概率并不超过50%，我在防5浪，但并不确定5浪一定会来。这是一个逻辑问题，我做的很多防守，当它不出现的时候，我们感受不到这么做的意义。"（2019）

> "我防5浪下跌并不代表了5浪下跌一定会来。我甚至明确表述过5浪下跌的概率并不超过50%，只不过依旧有防的必要。"（2019）

### 同一段行情能划出两种浪型

> "浪型上60%的概率是1浪上升……另外40%的概率是2浪反弹……这两种浪型划分的概率最大。那么我们怎么能量化到底是哪一种呢？"（2019）

60% vs 40%——他自己在同一时间对同一段行情有两种竞争性浪型标注，且无法当场判定哪一个是正确的。

### "有没有5浪，其实也不关键"

> "12345的浪型后面到底有没有5，其实也不关键，关键是战略上要有做多的勇气。"（2019）

> "判断这里是不是4浪的高点其实没有那么重要，因为有没有5浪的踏空和套牢两种风险的控制的位置关键不在这，而是在3浪的低点。"（2019）

### 浪型事后才能确认

> "从确定性的角度，4浪高点我给你找的点是当时最合理的点。可是如果不从确定性的角度，而是从逻辑的角度，判断这个4浪高点正确与否，却需要后面的走势来确定。而在我判断4浪高点之后的走势，目前到现在还无法确定这件事。"（2019）

> "只有创新低，才能确定是C浪，不创新低之前，都不能确定，只能说是概率增加而已。"（2019）

### 他曾中途推翻自己的浪型划分

> "后来市场持续的下跌，完全超过了ABC的范畴，C浪更快并且时间更长（第一波下跌8天，第二波下跌了20天），所以我认为浪型不是ABC，而是12345的概率是更大的。"（2019）

先划 ABC，行情走了一段后发现不对，改成 12345。他自己承认浪型是会"变"的——今天划的，明天可能推翻。

## 八、与时间对称的关系总结

```
波浪理论（上游框架）
    │
    ├→ 定义浪型级别 → "这是同级别的两段"
    ├→ 提供中轴来源 → 前一段的同级波段
    └→ 时间对称（下游应用）
            │
            ├→ 1浪≈5浪、2浪≈4浪
            ├→ A浪≈C浪
            └→ 跨段直接对称
```

徐小明的时间对称无法独立于波浪理论存在——"同级波段"的定义来自浪型标注。而浪型标注本身就是主观的。这就是时间对称量化失败的根源：**链条上游的主观判断无法被编码，下游的对称检测就没有可靠的输入。**

## 不采纳声明

波浪理论不进入正式分析框架，原因：

1. **徐小明本人的定位**："浪型不是规则"——他自己在交易时不等浪型确认
2. **确认滞后性**：3浪确认需要等到价格突破3674——等确认完行情已经过半
3. **量化路径不明**：浪型划分的多版本竞争和事后确认特性，决定了它在实时的自动化信号生成中不具备可操作性
4. **与时间对称的链路关系**：如果波浪理论本身不能量化，依赖它的时间对称也不能量化——这是系统性约束，不是单一工具的缺陷

本文档保留为附录，供后续使用本 skill 的人参考。如未来发现可编码的浪型标注方法，波浪理论和时间对称均具备重新评估的条件。
