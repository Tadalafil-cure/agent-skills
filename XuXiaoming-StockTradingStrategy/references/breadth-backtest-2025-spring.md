# NH/NL 广度指标回测：2025年春节前后

> 验证徐小明「春节长假之后的四个交易日，个股都是明显的强于大盘」与 NH/NL 指标的一致性。

## 结论

NH/NL 比涨跌家数更可靠——不会被单日情绪骗进去。叠加三层信号（指数裁决 + NH/NL广度 + 共振）优于单层。

## 核心发现

1. **2/5 NH/NL说"弱"但涨跌家数说"强"**（3483:1850）。NH/NL看60日新高新低——节后首日涨是涨了，但创60日新高的不多。这是超跌反弹不是趋势反转。

2. **1/9孤立试探被过滤**：裁决说"试探"，NH/NL说"底背离"（不是切换确认），共振不支持。1/15退回观望。NH/NL层有效阻止了假启动。

3. **2/7-2/12三层共振确认**：裁决试探→持股 + NH/NL切换确认 + 共振混合→偏共振→强共振。三层同时亮，这次没退回。

4. **NH/NL的"钝"是设计意图**：比单日涨跌家数慢5-7天确认，但过滤了假信号。2/5的假强势被正确忽略，2/12真反转才确认。
