## 一、市况判断（双维度并行）

### 维度1：三合一（方向）

```
ER30 + ADX(14) + 标准化动量(20)
→ 250天滚动65%分位 → p90归一化 → 连续趋势度 T

T > 0.6 → 上行趋势
T > 0.3 → 偏多
其他    → 震荡
T < -0.3 → 偏空
T < -0.6 → 下行趋势
```

### 维度2：CHOP(14)（强度/衰竭）

```
CHOP < 38.2  → 趋势行进中
CHOP 38-62   → 中性
CHOP > 61.8  → 衰竭/震荡
```

### 市况矩阵

| 三合一 \ CHOP | <38.2 趋势中 | 38-62 中性 | >61.8 衰竭 |
|:---|:---|:---|:---|
| **上行/偏多** | 强趋势涨 → 满仓持股 | 涨势减速 → 持股 | 上涨衰竭 → 持股警戒 |
| **震荡** | —（震荡少见低CHOP） | 标准震荡 → 观望 | 横盘衰竭 → 等突破 |
| **偏空/下行** | 强趋势跌 → 空仓 | 跌势减速 → 空仓+关注底 | 下跌衰竭 → 等底部信号 |
