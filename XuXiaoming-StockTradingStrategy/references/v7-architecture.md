# 裁决引擎 v7 架构方案

## 总架构

```
L1  月周九转   → 大周期定调（40/20天窗口，底部结构触发时升档试探→持股）
                不独立发信号，只做升档器

L2  日线层
  ├─ L2-1  多指数共振（优先确认层）
  │        五指数（上证/深成指/创业板/沪深300/中证500）通道状态
  │        强共振(≥4同向) → 趋势确认 → 提高后续信号置信度
  │        分化/混合      → 趋势存疑 → 单指数各自裁决
  │
  ├─ L2-2  上证指数（独立输出）
  │        三合一 + CHOP → 趋势/震荡
  │        ├─ 趋势期: 趋势为王，结构修边
  │        └─ 震荡期: CHOP变动 + 结构累计 → 独立发动
  │
  └─ L2-3  深证成指（独立输出）
           同上
```

**指数选择规则：** 2024年前以深证成指为主，2024年起兼顾上证（徐小明转上证）。

---

## L1: 月周九转（升档器）

| 信号 | 窗口 | 作用 |
|------|:--:|------|
| 月线低9 | 形成后40天 | L2 试探→升档为持股 |
| 周线低9 | 形成后20天 | L2 试探→升档为持股 |

不独立发信号。多层窗口叠加不叠加升档（最高到持股）。

---

## L2-1: 多指数共振（趋势确认层）

**数据源：** 五指数（上证/深成指/创业板/沪深300/中证500）的三合一方向

**共振等级：**

| 共振状态 | 定义 | 含义 | 对后续影响 |
|---------|------|------|----------|
| 强共振_上升 | ≥4指数上行/偏多 | 趋势确认 | 趋势信号置信度+1 |
| 强共振_下跌 | ≥4指数下行/偏空 | 趋势确认 | 趋势信号置信度+1 |
| 偏共振 | 3指数同向 | 趋势存疑 | 降级参考 |
| 混合/分化 | 方向不一致 | 无共识 | 转单指数独立判断 |

**关键规则：** 强共振时，单指数的方向判断不能推翻共振——即使深证判"震荡"，五指数强共振上升→仍按偏多处理。分化时反之，单指数可独立发动。

---

## L2-2/L2-3: 单指数日线裁决

每个指数独立运行，结构相同：

### 工具清单

| 工具 | 作用 | 参数 |
|------|------|------|
| **三合一** | 方向（上行/偏多/震荡/偏空/下行） | ER30 + ADX14 + 动量20，250天65%分位 |
| **CHOP(14)** | 趋势可信度 + 趋势升降 | <38.2 清晰, 38-62 模糊, >61.8 混乱 |
| **MACD结构** | 底部/顶部钝化→形成→消失 | 参数(4,30,4) |
| **日线九转** | 高9/低9 辅助信号 | 收盘价 vs 4根前收盘价 |

### 趋势期 vs 震荡期（三合一 + CHOP 联合判定）

**判定规则：**

```
三合一 = 上行/偏多 + CHOP < 61.8 → 趋势_上升
三合一 = 下行/偏空 + CHOP < 61.8 → 趋势_下跌
三合一 = 震荡    + CHOP < 38.2 → 按bullish方向归入趋势（偏多/偏空）
其他                              → 震荡
```

**CHOP > 61.8 强制归入震荡**——无论三合一说什么，趋势已不可靠。

---

### 趋势期的裁决

```python
# 趋势上升
if 三合一 == 上行/偏多:
    基础 = 持股

    # 顶部结构修边
    if 顶部结构形成（含前5天）:
        if CHOP < 38.2:                      → 持股(警戒)    # 趋势清晰，只加标记
        elif CHOP 38-62:                     → 持股(警戒)    # 趋势模糊，警戒升级但不动
        # CHOP>61.8 不在此（已归入震荡）

    # 序列辅助
    if 日线高9:
        if CHOP < 38.2:                      → 持股(警戒)
        elif CHOP 38-62:                     → 持股(警戒)

    # 多指数共振确认
    if 强共振_上升:                          → 置信度+1
    elif 分化:                               → 置信度-1（等单指数确认）

# 趋势下跌
if 三合一 == 下行/偏空:
    基础 = 空仓

    # 底部结构：可独立入场（徐小明核心规则）
    if 底部结构形成 + 综合筛选通过:
        → 试探(恐慌底/磨底/衰竭底)

    # 序列辅助
    if 日线低9:                               → 空仓(关注)

    # 多指数共振
    if 强共振_下跌:                           → 空仓置信度+1
```

### 底部结构综合筛选（入场条件）

**不是 bs=1 就触发。必须同时满足：**

| 条件 | 阈值 | 说明 |
|------|------|------|
| **60日回撤** | > 15% | 跌得够透 |
| **结构拥挤度** | 前180天 ≤ 1个底结构 | 不拥挤，信号干净 |
| **钝化质量** | 自上次结构后钝化 2-3 天 | 非瞬发(0-1天)非钝而不化(>5天) |
| **+ CHOP 模式** | 见下方 | 恐慌底/磨底/衰竭底三选一 |

**CHOP 模式分类（三选一，满足任一即触发）：**

```
恐慌底:  CHOP曾>60(前30天) + 当前CHOP<40  → "衰竭→恐慌出清→反转"
磨底:    CHOP 38-62（当前）               → "阴跌磨底→最后一脚"
衰竭底:  CHOP > 61.8（当前）              → "极度衰竭→物极必反"
```

**月/周低9窗口升档：** 若在窗口内，试探→升档为持股。

---

### 震荡期的裁决

```python
if 震荡:
    # 结构信号在震荡期权重降低，但可以独立发动
    if 底部结构 + 综合筛选通过:
        → 试探(磨底/衰竭底)    # 震荡+结构=试探

    if 顶部结构形成:
        → 空仓                # 震荡+顶结构=走

    # CHOP 变动：趋势突破前兆
    if CHOP 持续下降(前10日均 > 当前 + 差值>5):
        if 三合一方向转 bullish:
            → 试探            # CHOP收敛+方向转正→等突破
            # 若叠加强共振上升 → 升级为持股

    # 序列
    if 日线高9: → 观望(偏空)
    if 日线低9: → 观望(偏多)
```

**震荡期的核心逻辑：** 趋势不明确→观望为主。但底部结构综合筛选通过+CHOP模式匹配→独立入场。这是徐小明"要么突破趋势要么底部结构形成"中"底部结构形成"的那条路。

## 输出格式

每个交易日一行，包含：

| 字段 | 来源 | 说明 |
|------|------|------|
| date | — | 日期 |
| close_sz / close_sh | OHLC | 收盘价 |
| regime_sz / regime_sh | 三合一 | 方向 |
| chop_sz / chop_sh | CHOP | 可信度 |
| chop_level | 分级 | clear/fuzzy/chaotic |
| resonance | 共振 | 强共振/偏共振/混合/分化 |
| bs / ts | 结构引擎 | 底/顶结构形成 |
| bs_filter | 综合筛选 | 通没通过 |
| bs_pattern | CHOP模式 | 恐慌底/磨底/衰竭底 |
| day_seq | 序列引擎 | 高9/低9 |
| month_win / week_win | 月周窗口 | 是否在窗口内 |
| verdict_sz | 深证裁决 | 持股/试探/空仓/观望 |
| verdict_sh | 上证裁决 | 同上 |
| verdict_final | 综合裁决 | 按指数选择规则合并 |
| reason | — | 裁决理由 |

**指数选择规则（verdict_final）：**
- 2024年之前：以深证 verdict_sz 为准
- 2024年起：上证权重50% + 深证权重50%，冲突时取更保守的

## 与徐小明原文的对应关系

| 徐小明表述 | 引擎对应 |
|-----------|---------|
| "趋势为王" | 三合一方向 + CHOP 可信度 + 多指数共振确认 |
| "结构修边" | 顶部结构在趋势中只加警戒、在模糊时升级警戒 |
| "底部结构形成可入场" | 综合筛选(回撤+拥挤+钝化+CHOP模式)通过 → 试探 |
| "趋势突破可入场" | CHOP收敛 + 三合一方向转正 → 试探/持股 |
| "序列不是交易规则" | 日线九转只加减标记(警戒/关注)，不独立发信号 |
| "序列的缺点就是丢趋势" | 高9+趋势上升→持股(警戒)，不推翻趋势 |
| "月线低9战略建仓" | 底部试探 + 月低9窗口 → 升档为持股 |
| "多看少动/随缘" | 震荡+无结构+无CHOP变动 → 观望 |
| "上证为标准"(2024后) | 2024起上证权重加入 verdict_final |

## 盲测验证与已知局限

> 2026-07-12 · 两轮LLM盲测（v14: 2020-2022 90篇, v15: 2023-2024 60篇）

### 盲测结果汇总

| 盲测 | 年份 | 一致 | 部分 | 不一致 | 加权率 | 核心教训 |
|------|------|:--:|:--:|:--:|:--:|------|
| v14 | 2020-2022 | 21 | 27 | 42 | **37.8%** | 2022年仅13.3%——v7全年空仓vs XXM9月战略性看多 |
| v15 | 2023-2024 | 23 | 27 | 10 | **60.8%** | 2024年66.7%优于2023年55.0%；结构信号日反比非结构日差 |

### 按市况分层

| 市况 | v14+v15样本 | 加权率 | 问题 |
|------|:--:|:--:|------|
| 上行趋势/偏多 | ~15篇 | **~72%** | 趋势清晰时v7表现良好 |
| 震荡 | ~35篇 | **~65%** | 震荡中v7倾向观望，徐有时更积极 |
| 偏空/下行趋势 | ~25篇 | **~40%** | **核心失分项**——v7在下行趋势中过度保守 |

### 三大系统性问题

#### 1. BS综合筛选过于严格（最高优先级修复）

**症状**：2023-12-27徐小明明确宣告"日线底部结构形成"（四指数日线结构+五指数周线钝化+四指数月线序列），v7当日判空仓。

**根因**：四条件AND逻辑（回撤>15% + 拥挤度≤1 + 钝化2-3天 + CHOP模式），2023年底部经过反复钝化/形成/消失后拥挤度超过阈值。徐小明本人是在结构"最终形成"时入场，不是等"干净"结构。

**建议修复**：
- 月/周低9窗口内降为四选二
- 拥挤度改为"前180天≤3"（当前≤1过严）
- 钝化天数放宽到1-5天（覆盖反复钝化场景）

#### 2. 下行趋势过度保守

**症状**：15篇下行趋势样本加权率仅43%。v7坚持空仓，拒绝所有底部试探。

**根因**：当前规则"趋势向下→空仓→底部结构+BS筛选→试探"，但BS筛选本身已足够保守（触发率5%），叠加趋势向下形成双重过滤。

**建议修复**：
- 月/周低9窗口内：趋势向下+底部结构形成→跳过BS筛选，直接试探
- 徐小明"月线低9...这种建仓是具有战略意义的"——v7当前在H10中有此规则，但实现时BS筛选阻塞了通道

#### 3. 924后高位震荡v7过于乐观

**症状**：2024年10-12月徐多次表达"窄幅震荡/消耗时间/多看少动"，v7在强共振_上升下持续判持股。

**根因**：强共振_上升的确认信号在暴涨后形成惯性，三合一方向仍判trend→持股。但徐小明在3浪快速拉升后的横盘期会主动切换为"等方向"。

**建议修复**：
- 强共振上升后若5日涨幅<1%+CHOP上升→降为观望
- 增加"高位横盘"识别：前20日曾涨>15% + 近10日涨<2% → 震荡优先

### 友好率（放宽判定）

若将"部分一致"视为可接受（方向不矛盾即友好），v7整体友好率为：
- 2023-2024: (23+27)/60 = **83.3%** — 大部分分歧是程度问题非方向问题
- 2020-2022: (21+27)/90 = **53.3%** — 2022年存在实质方向矛盾
