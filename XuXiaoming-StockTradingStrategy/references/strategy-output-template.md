## 三、趋势-结构-序列分析推演

> 核心原则：趋势 > 结构 > 序列，不可颠倒。
> 每层分析必须引用徐小明原文（格式：`[YYYYMMDD] 原文摘录`）。

### 趋势

1. 先说三合一方向 + CHOP 可信度，判断当前是否可执行趋势策略
2. 引用原文支撑：

> [20190210] 但如果突破趋势，你就甭管结构了。**趋势为王，结构修边。**

3. 若趋势不可靠（震荡/CHOP高），引用：

> [20191031] 我说这里的行情很难做，是因为**震荡趋势挨打，震荡的过程中高低点又没有结构**。

> [20210824] 震荡行情的特征就是涨涨跌跌，而在涨跌的过程中，会**反复的上穿趋势和破位趋势**。

### 结构

1. 过去一段时间的结构事件（顶结构触发什么操作、底结构有无）
2. 当前 MACD 钝化状态（无钝化/底钝化/顶钝化）
3. 引用原文：

> [20190812] 买入的原则要符合交易的原则，要么突破趋势，要么底部结构形成。**序列不是买入原则，结构和趋势才是。**

4. 若有底部结构，引用恐慌底/磨底相关原文（见`references/original-text-dual-pattern.md`）：

> [20240709] 钝化是一个过程。从钝化到结构形成，是逐渐确定的过程。**不是有了钝化就一定形成结构，而是要经历这个过程。**

### 序列

1. 月/周/日线九转状态
2. 引用原文：

> [20190804] 但**序列并不是交易标准**，如果你按序列卖出，能卖在最高点上，可是这样的交易对标准来讲就是一种伤害。

> [20200513] **序列的缺点就是丢趋势**，一个趋势性的行情，序列几乎都是不对的。

3. 若震荡市中序列有效，引用：

> [20220929] **月线的低9是一个月线时间点，前两次月线低9都筑大底了。**

### 综合分析

三层交叉验证结论：趋势给什么方向、结构给什么信号、序列给什么辅助——三者是什么关系（同向共振/互相矛盾/单层有信号）。引用：

> [20190731] 共振的方式有两种，第一种是**多周期的共振**，小周期是高点，大周期也是高点。**大小周期重叠在一起，产生共振的作用，这个时候可以判定高点的级别比较大。** 第二种是**多品种共振**，指数之间有共振作用。

### 分钟线（v4.4.1 新增）

1. 报告最近 2-3 天的分钟线结构状态（S/A/B/C 级、顶/底方向、周期数）
2. 判断市况是否允许分钟线触发（震荡+fuzzy → 禁用）
3. 若允许，检查日线确认条件（低9/bs/强共振下跌/急跌）
4. 结论：分钟线是否被沉默、或是否触发升级
5. **若模型沉默但徐小明原文会触发，加脚注**：⚠️ 徐小明原文对此的处理：[简述]。模型回测数据：[N]次同类信号，胜率[XX]%，均收益[XX]%。当前按模型规则执行。参见 `references/model-vs-xuxiaoming-divergence.md`。

## 六、总结

严格使用徐小明公众号写作风格：

**风格要求：**
- 语调：直接、指导式——不学术、不文艺、不模棱两可
- 视角："我们"——始终和读者同一战线
- 段落：短段落（3-5句），从不堆砌
- 结构：判断→操作→纠错→信念收尾
- 用词：时期三（2024-2026）风格——极端短句，少论证"为什么"，直接给"是什么"
- 独有短语：按语境选择——「趋势之上持股，破趋势减仓」「化繁为简」「随缘」「多看少动」「按规则来」「等着就好」
- 结尾必有信念句——「市场会给答案的」「规则就是规则」「按规则踏踏实实的做就行了」

**内容要点：**
1. 当前判断（≤3句，给结论不给分析过程）
2. 操作（≤2句，明确做/不做）
3. 纠错标准（≤2句，什么情况下推翻当前判断）
4. 信念收尾（1句）

**示例：**

```
当前判断：

深证成指日线低9，序列值9。序列的缺点是丢趋势，但现在是震荡市，序列在震荡市里比趋势市里好用。

日线趋势在震荡里，没有方向。结构也没有——从6月22日16374高点下来，MACD还没出底部钝化。所以序列给了信号，结构不给确认。

上证比深证稳一点，从5月14日4259高点下来跌了5%，深证跌了8%。但上证没有序列低9，也没有结构。

两个指数都没到交易标准。

操作：

观望。多看少动。

不是我们不想做，是没有符合规则的地方做。等MACD出底部钝化，等底部结构形成，形成之后再定。深证已经有低9了，如果出结构，是序列加结构的配合，可以重视。

纠错标准：不跌反涨，等趋势突破再纠错买回——趋势突破是唯一的上车标准，不追。

等着就好，市场会给答案的。
```

## 七、极简版（时期三风格 · 与原文对照用）

> **本节将前六段的模型判断压缩为时期三极简风格，用于和徐小明原文的风格级比对，不替代前六段。**

**风格要求：**
- 零技术术语 — 不提趋势/CHOP/三合一/结构/序列/MACD/钝化
- 600 字以内，短段落（3–5句/段）
- 2 步论证：判断 → 结论，不铺陈"为什么"
- 视角："我们"，不用"模型""引擎""裁决"
- 结构：当前判断 → 操作 → 纠错标准 → 信念收尾

**必需内容要点：**

| 要点 | 对应前六段来源 | 处理方式 |
|------|-------------|---------|
| 指数区间判断 | 一·当前市况 + 二·年线回溯 | 化为日常语言（如"3800-4200来回走"） |
| 方向偏斜 | 三·趋势分析 + 四·裁决 | 有就说、无就坦言"看不清" |
| 结构有无 | 三·结构分析 | 有结构说标准，没结构说"没标准就不做" |
| 操作 | 四·合并裁决 | 直接一个字：持/等/空/试/减 |
| 纠错标准 | 四·裁决推演 | 什么情况推翻当前判断 |
| 信念收尾 | 六·总结 | 一句 |

**示例：**

```
当前判断：

指数在3800到4200之间来回走，走了快半年了。这个震荡没什么要结束的意思，往上往下空间都不大。短期方向看不清。

深证有个低9，但序列不是交易标准。震荡市里可以看一看，不足以动手。结构也没出来——两个指数都没有底部钝化，没结构就是没标准，没标准就不做。

操作：

观望。多看少动。

不是说行情没机会，是有机会的地方不在我们的规则里。趋势没方向，结构没信号，就等。等底部钝化出来，等钝化之后形成结构，那时候再说。

纠错标准：不跌反涨，等趋势突破追。不突破不动。

等着就好，市场会给答案的。
```

**与原文比对要点：**

比对时关注三个层次，不只看表面对错：
1. **方向**：看多/看空/观望是否一致（不一致才标红）
2. **强度**：乐观 vs 保守（强度差异不是方向矛盾，按 A10 处理）
3. **分析维度**：技术在说技术信号、市场在说市场风格（维度不同不判对错，标注即可）
