# 裁决偏多标签来源追踪

> 2026-06-25 会话产物。防止将 verdict_main/verdict_tech 的「偏多」后缀误解为多指数合力。

## 问题

当 verdict_main = 观望(偏多) 且 五指数共振 = 混合（全震荡）时，「偏多」的底气从哪来？

## 偏多后缀的四条来源路径

| 路径 | 规则 | 触发条件 | 性质 |
|------|------|------|------|
| 低9 | R4.3 | 震荡 + 日线低9 | 单个指数的序列关注偏置 |
| 来路上行 | H17 | 震荡 + 来路 ≥7天上行趋势 | 历史趋势的方向惯性 |
| CHOP收敛 | R2.4 | 震荡 + CHOP<38.2 持续≥3天 + 方向偏多 | 趋势涌现的预先偏置 |
| 强共振上升 | R5.1 | ≥4 指数同向 | **真正的多指数合力** ← 最可信 |

## 7/13 案例

```
五指数共振: 混合（上证/深证/创业板/沪深300/中证500 全震荡）
verdict_main: 观望(偏多) ← 来源：深证 低9（R4.3）

逐指数拆解：
  上证   观望      (无偏多标签)
  深证   观望(偏多) → 震荡+低9
  沪深300 无独立裁决
  中证500 无独立裁决
  → 五指数中仅深证有偏多，且来自序列而非趋势
```

## 解读规则

1. 展示裁决时必须标注偏多后缀的**具体来源**（震荡+低9 / 来路上行→偏多 / CHOP收敛）
2. 五指数共振=混合时，偏多≠共振背书，只是单指数信号
3. 主板/科技裁决取强时，若两指数 signal_strength 平手，按代码默认取前一个（SZ/CYB）——这可能导致偏多标签被高估
4. 沪深300/中证500 参与共振计算但无独立裁决，分析五指数时需额外拉取
