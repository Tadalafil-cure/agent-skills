# 盘后全流程执行清单

每次生成操作策略报告时，按此清单顺序执行，缺哪步停哪步。

```
-1  日期确认       date "+%Y-%m-%d %A"
 0  市场状态       market_status() → 盘前/盘中/盘后
 1  数据管线       build.py（日线+分钟线全量，含 fetch_all_minute，已固化v4.6.23）
 2  管线验证       验证 7+4 文件产出（日线/结构/序列/裁决 + 四指数分钟线）
 3  交叉验证       日线|结构|序列|裁决|分钟线四指数|五指数投票明细
 4  主Agent统稿    一~八节 ⛔不碰博客·不拆子Agent
 5  **校验Agent**  逐条对照关键规则审查一~八节定稿
 6  博客Agent      A20隔离·**一~八节 write_file 完成后再跑** blog_monitor.py
 7  追加比对       仅第九节·不修改一~八节
 8  清理           旧报告删除
 9  A20扫描        python scripts/check_a20.py reports/<新报告>.md  ← 交付前最后一步
                    检查：A20隔离·震荡来路·v4.6.23术语·共振投票表·NH/NL·分钟线
```

## 关键规则

- **不拆子 Agent 做七层递进**：主 Agent 自己读数据、自己推演、自己写。
- **博客后置**：主 Agent 写完一~八节后，才派独立子 Agent 读博客。
- **共振投票表必写**：报告第一节必须包含五指数共振投票明细表。
- **钝化回测结论必嵌入**：分钟线底钝化单用不可靠（假信号46-100%），必须结合日线低9/CHOP。
- **CHOP 分级铁律**：<38.2=clear，38-62=fuzzy，>61.8=chaotic。
- **A19 分列**：顶/底结构按市况分开统计。

- 🔴 **v4.6.24 震荡框架（H17——入震退震全链）**：
  入震：三合一≥2d 滞回（上下行统一）
  退震：按来路分流——上行来路≥3d / 下行来路≥4d
  震荡中：⛔ 不操作（规则层），BS信号→推论层 S4/S5
  CHOP=辅助烈度，方向为主
  退震确认前不跟方向，所有试探信号须抑制
  退震确认后：上行向上CHOP<38强持(20d+3.1%胜率91%)，CHOP>50减仓；向下全域该砍
  下行向上全域正收益（反转信号≈+2~5%）；下行向下全域该砍（中继信号，方向弱偏不预判）
  四指数来路一览表必须含退震状态和确认天数

- 🔴 **A20 博客隔离——时序硬约束（v1.14）**：
  blog_monitor.py 必须在一~八节 write_file 完成后再调用
  违规特征：一~八节出现当天博客时效内容（「徐小明今天说」「今日博文指出」）
  **引用徐小明历史原文（方法论/规则，如「趋势为王结构修边」）不受此限**
  已犯5次（7/10+7/14+7/16+7/20初版+7/20修复版），根因：blog_monitor 和分析数据同轮拉取→博客渗透分析段
  ⚠️ delegate_task 子Agent也须遵守：子Agent上下文中的A20约束可能被自主工具调用顺序破坏
    修正：子Agent prompt中须写明「先 write_file 一~八节，最后再 terminal blog_monitor」的硬顺序

- 🔴 **日期校验必须用 `date` 命令**：禁止凭上下文推断日期。

- 🔴 **A20 执行隔离——引擎数据与博客分两轮调用**：`blog_monitor.py` 不得与引擎数据查询出现在同一个 `terminal` 调用或 `&&` 链条中。第一轮只拉引擎数据→写完一~八节→第二轮独立运行 `blog_monitor.py`→追加第九节。

- ⚠️ **周五博客偏移（v4.6.20 修正）**：D=周五时，先查 `blog_monitor.py --date`（周五→「周一操作策略」），查到就用；查不到才回退周四。

- 🔴 **H17b 上行来路底结构抑制**：震荡来路=上行时，底部结构不触发试探/持股——仅关注。

- 🔴 **M1 规则独立——CHOP 回落 ≠ M1 解锁**：CHOP 越高分钟线底结构越可靠，CHOP 越低噪音率 29% 反而禁用。

- 🔴 **A18 分钟线强制报告**：四指数分钟线状态必须出现在报告二/三节推演中 + 四节跨市场交叉分析汇总。

- 🔴 **A18e 钝化数据必嵌入**：即使当前无钝化（bd=0），也须标注「无钝化」并引用回测基线。

- 🔴 **A19b 统计分开 ≠ 思维分开**：表格分开后，后续论证必须锚定切分后的子数。

- 🔴 **A23 五指数共振成员固定**：科创50明确排除，仅参与科技板双创共振。

- 🔴 **A24b 来路独立计算——禁止跨指数同步**：每个指数的来路由自身 regime 历史独立判定，互不同步。

- 🔴 **市况定性闸门（v1.13）**：「趋势向上/向下已确认」须同时满足三条件（下行趋势+≥3天+共振≥2天）。落闸口诀：「偏空不是下行趋势，首日不称已确认」。

- 🔴 **文件名规范**：`reports/YYYY-MM-DD_主题_YYYYMMDD_HHMMSS.md`。
