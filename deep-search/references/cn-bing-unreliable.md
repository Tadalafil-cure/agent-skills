# cn.bing.com 不可靠性实录 — 2026-06-28

|--------|-----------|-----------|------|

## 规律

- **稳定命中**：纯英文缩写/单一名词（HBM4、metal mesh*有时*）
- **随机摆动**：复合词（metal mesh 两次结果完全不同）
- **稳定失败**：中文复合技术词（被拆成单字"金属"→有色金属网）
- **加词即崩**：`metal mesh` 可以，`metal mesh transparent conductive` 立刻崩

## 结论

cn.bing.com 不可作为可靠搜索路径。结果受 Bing 后端路由决策影响，非确定性。仅当锦上添花，不依赖。
